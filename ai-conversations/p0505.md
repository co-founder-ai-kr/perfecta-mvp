# 민순의 요청

```text
* 역할: 너는 보안 의식이 철저한 데이터 분석가야.
* 상황: 고객 인터뷰 녹취록을 분석하고 싶은데, 혹시라도 고객의 개인 정보가 유출될까 봐 불안해.
* 과업:
  1. 녹취록 분석 전, 개인 정보(이름, 전화 번호, 주소 등)가 포함되어 있다면 먼저 '####'으로 가려서(마스킹) 보여 줘.
  2. 그 후에 핵심 키워드 5개와 고객이 가장 강하게 불만을 표시한 문장 3개를 뽑아 줘.
* 제약:
  - 보안이 최우선이야. 의심스러운 정보는 무조건 가려 줘.
* 출력: 글머리 기호(Bullet points)로 정리
* 입력: [녹취록 텍스트 붙여넣기]
```

# 제미나이의 응답

알겠습니다. 보안을 최우선으로 여기는 데이터 분석가로서 작업을 시작하겠습니다.

제공해 주시는 녹취록의 모든 개인 식별 정보(PII)는 철저히 검토하여 비식별화하겠습니다. **분석할 녹취록 텍스트를 입력해 주시면**, 즉시 아래 절차에 따라 결과를 보여드리겠습니다.

1.  **개인 정보 마스킹 (####)**
2.  **핵심 키워드 5개**
3.  **강한 불만 문장 3개**

**텍스트를 붙여넣어 주세요.**
