# 민순의 요청

```text
* 역할: 너는 꼼꼼한 회의 서기이자 상품 기획자야.
* 상황: 도예가와의 미팅 녹취록을 보고 핵심 내용과 할 일을 정리하고 싶어.
* 과업:
  1. 작가의 철학이랑 공급 가능 수량을 3줄로 요약해 줘.
  2. 내가 이번 주에 처리해야 할 일을 뽑아 줘.
  3. 액션 아이템은 시스템이 인식할 수 있도록 JSON 형식으로 만들어 줘.
* 출력:
  1. 핵심 요약 (텍스트)
  2. 액션 아이템 (JSON 형식)
* 입력: [녹취록 텍스트 붙여넣기]
 (예시:
    도예가: 제 작품은 자연의 불규칙한 아름다움을 담고 있습니다. 한 달에 10개 정도 공급 가능합니다. 
    민순: 그럼 다음 주까지 샘플 3개 받아볼 수 있을까요?)
```

# 제미나이의 응답

네, 꼼꼼한 회의 서기이자 상품 기획자로서 도와드리겠습니다.

현재 **[녹취록 텍스트]**가 입력되지 않았습니다. 분석이 필요한 **녹취록 전문을 붙여넣어 주시면**, 요청하신 대로 완벽하게 정리해 드리겠습니다.

우선, 입력해주신 **예시 텍스트(도예가와 민순의 대화)**를 바탕으로 결과물이 어떻게 나오는지 보여드리겠습니다.

---

### [예시 데이터 기반 분석 결과]

#### 1. 핵심 요약 (텍스트)
1. 작가는 인위적이지 않은 **'자연의 불규칙한 아름다움'**을 작품의 핵심 철학으로 삼고 있음.
2. 수작업 공정 특성상 **월 공급 가능 수량은 약 10개** 내외로 한정적임.
3. 상품성 검토를 위해 **다음 주까지 샘플 3개**를 수령하기로 협의함.

#### 2. 액션 아이템 (JSON 형식)

```json
{
  "action_items": [
    {
      "id": "AI-001",
      "category": "Sample Review",
      "task": "도예가로부터 샘플 3개 수령 및 퀄리티 확인",
      "assignee": "민순",
      "deadline": "Next Week",
      "priority": "High",
      "status": "Pending"
    }
  ]
}
```
