```text
* 역할: 너는 구글 서치 콘솔 전문 튜터야.
* 상황: 개발 지식이 없어서 '구글 서치 콘솔' 등록이 너무 무서워. DNS가 뭔지도 몰라.
* 과업:
  1. 구글 서치 콘솔 사이트 접속부터 소유권 확인까지의 절차를 단계별로 설명해 줘.
  2. 소유권 확인을 위해 필요한 HTML 태그를 어디에 붙여넣어야 하는지 알려 줘.
* 제약: 
  - DNS 설정 같은 어려운 방법은 제외해 줘.
  - HTML 태그 인증 방식으로 코드 복사 붙여넣기로 끝나는 가장 쉬운 방법을 안내해 줘.
```

# 제미나이의 응답

안녕하세요! 구글 서치 콘솔 튜터입니다. 👋
개발 지식이 없어도, DNS가 뭔지 몰라도 전혀 상관없습니다. **"복사하고 붙여넣기"** 딱 두 가지만 할 줄 알면 누구나 할 수 있어요.

가장 쉬운 **'HTML 태그'** 방식을 이용해서, 아주 천천히 단계별로 알려드릴게요. 겁먹지 말고 저만 따라오세요!

---

### 1단계: 구글 서치 콘솔 접속 및 로그인

1.  **[구글 서치 콘솔](https://search.google.com/search-console/about)** 사이트에 접속하세요.
2.  **'시작하기'** 버튼을 누르고, 평소 사용하는 구글 아이디로 로그인하세요.

### 2단계: 속성 유형 선택 (🚨 여기가 제일 중요해요!)

로그인을 하면 화면에 두 개의 큰 박스가 나옵니다.
왼쪽은 '도메인', 오른쪽은 'URL 접두어'라고 되어 있을 거예요.

*   **왼쪽(도메인)은 절대 건드리지 마세요!** (이게 바로 어려운 DNS 설정입니다.)
*   **무조건 오른쪽 [URL 접두어] 박스를 선택하세요.**

1.  **URL 접두어** 박스 입력창에 내 사이트 주소를 **전체 다** 입력하세요.
    *   예시: `https://www.mysite.com` (앞에 https:// 까지 꼼꼼하게!)
2.  **'계속'** 버튼을 클릭하세요.

### 3단계: 인증 방법 선택 (HTML 태그)

잠시 기다리면 '소유권 확인' 창이 뜹니다. 여러 가지 방법이 나오는데, 우리는 가장 쉬운 방법을 쓸 거예요.

1.  화면을 조금 내려서 **'HTML 태그'** 라는 항목을 찾아서 클릭(화살표)하세요.
2.  그러면 긴 영어 코드가 나옵니다. (`<meta name=...` 으로 시작하는 코드)
3.  그 옆에 있는 **[복사]** 버튼을 누르세요.
    *   *주의: 이 창을 닫지 말고 그대로 켜두세요! 코드를 붙여넣고 다시 돌아와야 합니다.*

### 4단계: 코드 붙여넣기 (어디에 넣나요?)

이제 복사한 코드를 내 홈페이지의 **머리(Head)** 부분에 붙여주면 됩니다. "구글아, 여기 내 명찰 붙였으니까 확인해 줘!"라고 하는 과정이에요.

사용하시는 블로그나 사이트 제작 도구에 따라 위치가 조금씩 다르지만, 원리는 똑같습니다. **`<head>` 와 `</head>` 사이**에 넣으면 됩니다.

가장 많이 쓰는 두 가지 경우를 예로 들어드릴게요.

#### A. 티스토리(Tistory)를 쓰는 경우
1.  티스토리 관리자 페이지로 갑니다.
2.  왼쪽 메뉴에서 **'꾸미기' > '스킨 편집'**을 누릅니다.
3.  오른쪽 상단에 **'html 편집'** 버튼을 누릅니다.
4.  영어 코드가 잔뜩 나오죠? 겁먹지 마세요. 위쪽에서 **`<head>`** 라는 글자를 찾으세요. (보통 3~5번째 줄에 있어요.)
5.  `<head>` 바로 아랫줄에 엔터를 쳐서 빈 줄을 만들고, 아까 복사한 코드를 **붙여넣기(Ctrl+V)** 하세요.
6.  **'적용'** 버튼을 누릅니다.

#### B. 워드프레스(WordPress)를 쓰는 경우
(플러그인을 쓰는 게 제일 안전하고 쉽습니다.)
1.  워드프레스 관리자 메뉴에서 **'플러그인' > '새로 추가'**로 갑니다.
2.  **'WPCode'** 또는 **'Insert Headers and Footers'**를 검색해서 설치하고 활성화하세요.
3.  왼쪽 메뉴에 생긴 **'Code Snippets' > 'Header & Footer'**로 들어갑니다.
4.  **Header**라고 적힌 빈칸에 복사한 코드를 **붙여넣기(Ctrl+V)** 하세요.
5.  **'Save Changes'**를 눌러 저장합니다.

### 5단계: 소유권 확인 완료

이제 다시 아까 켜두었던 **구글 서치 콘솔 화면**으로 돌아오세요.

1.  HTML 태그 항목 아래에 있는 초록색 **[확인]** 버튼을 누르세요.
2.  잠시 후... **"소유권이 확인됨"** 이라는 초록색 창이 뜨면 성공입니다! 🎉
3.  **'속성으로 이동'**을 누르면 끝입니다.

---

### 💡 튜터의 한마디
어때요? DNS 같은 어려운 용어 몰라도 "복사해서 `<head>` 밑에 붙여넣기"만 하면 끝납니다.

*   **혹시 실패했다고 뜨나요?**
    *   코드를 붙여넣고 '저장'이나 '적용' 버튼을 안 눌렀을 수 있어요.
    *   사이트에 반영되는 데 1~2분 정도 걸릴 수 있으니, 잠시 뒤에 다시 [확인] 버튼을 눌러보세요.

이제 구글이 님의 사이트를 잘 읽어갈 수 있게 되었습니다. 고생 많으셨어요!
