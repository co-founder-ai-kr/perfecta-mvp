# 제미나이의 응답

안녕하세요! 구글 서치 콘솔 튜터입니다. 👋

"DNS", "서버 설정"... 이런 단어들만 들어도 머리가 아프고 무서우셨죠? 걱정하지 마세요. 우리는 그런 복잡한 건 싹 다 무시하고, **가장 쉽고 직관적인 '복사+붙여넣기' 방법(HTML 태그 방식)**으로 딱 5분 만에 끝내볼 거예요.

심호흡 한번 하시고, 저만 믿고 천천히 따라오세요!

---

### 1단계: 구글 서치 콘솔 접속 및 속성 추가

가장 먼저 구글 서치 콘솔 사이트에 들어가서 "내 사이트 여기 있어요~" 하고 등록을 시작해야 합니다.

1.  **구글 서치 콘솔 접속:** [여기(Google Search Console)](https://search.google.com/search-console/about)를 클릭해서 접속하세요.
2.  **로그인:** 구글 아이디로 로그인하고 **[시작하기]** 버튼을 누릅니다.
3.  **속성 추가:**
    *   왼쪽 상단에 메뉴(햄버거 버튼 ≡)를 누르고, 목록 맨 아래 **[+ 속성 추가]**를 클릭하세요. (처음 방문했다면 바로 선택 화면이 뜹니다.)

### 2단계: 유형 선택 (★제일 중요★)

여기서 실수를 가장 많이 합니다. 화면에 두 개의 박스가 뜰 텐데, **무조건 오른쪽**을 선택해야 합니다.

*   **왼쪽 (도메인):** ❌ 선택하지 마세요! (이게 바로 그 무서운 DNS 설정이 필요한 곳입니다.)
*   **오른쪽 (URL 접두어):** ✅ **여기를 선택하세요!**
    *   입력창에 본인 사이트 주소를 `https://`를 포함해서 완벽하게 적어주세요.
    *   예: `https://www.mysite.com` 또는 `https://blog.naver.com/myid`
    *   다 적었으면 **[계속]** 버튼 클릭!

### 3단계: 인증 방법 선택 (HTML 태그)

이제 구글이 "이 사이트 진짜 네 거 맞아?"라고 물어보는 단계입니다. 여러 가지 방법이 나오는데, 우리는 **'HTML 태그'**를 찾을 겁니다.

1.  화면에 '권장 확인 방법'으로 'HTML 파일'이 뜰 수 있는데, 이건 무시하고 **아래로 스크롤**을 살짝 내리세요.
2.  **'다른 확인 방법'** 목록에 있는 **[HTML 태그]**라는 항목을 클릭해서 펼치세요.
3.  그러면 `<meta name="google-site-verification..." />` 처럼 생긴 긴 영어 코드가 보일 거예요.
4.  그 옆에 있는 **[복사]** 버튼을 누르세요. (이제 코드가 복사되었습니다!)

> **주의:** 아직 [확인] 버튼 누르지 마세요! 코드를 사이트에 붙여넣고 와서 눌러야 합니다. 이 창을 끄지 말고 그대로 두세요.

---

### 4단계: 코드 붙여넣기 (어디에 넣나요?)

이제 복사한 코드를 내 사이트의 **대문(Head)**에 붙여넣어야 합니다.

**📍 붙여넣어야 할 위치의 규칙**
HTML 문서에는 `<head>`라는 머리말 시작 태그와 `</head>`라는 머리말 끝 태그가 있습니다. **이 두 태그 사이** 아무 곳에나 붙여넣으면 됩니다.

#### 상황별 붙여넣기 방법

**A. 티스토리(Tistory) 블로그를 쓰신다면:**
1.  티스토리 관리자 페이지 접속 → 왼쪽 메뉴 **[꾸미기]** → **[스킨 편집]** 클릭.
2.  오른쪽 상단 **[html 편집]** 버튼 클릭.
3.  영어 코드들이 잔뜩 나오죠? 당황하지 말고 제일 위쪽을 보세요.
4.  `<head>`라는 글자가 보일 겁니다. 그 바로 밑에 엔터를 쳐서 줄을 만들고, 아까 복사한 코드를 **붙여넣기(Ctrl+V)** 하세요.
5.  오른쪽 상단 **[적용]** 버튼 클릭!

**B. 워드프레스(WordPress)를 쓰신다면:**
*   가장 쉬운 방법은 플러그인을 쓰는 것입니다.
*   'Insert Headers and Footers' 같은 플러그인을 설치하고, 설정 메뉴에서 **'Scripts in Header'** 칸에 복사한 코드를 붙여넣고 저장하면 끝입니다.

**C. 직접 만든 HTML 사이트라면:**
*   `index.html` 파일을 여세요.
*   `<head>` 태그를 찾으세요.
*   `<body>` 태그가 나오기 전, `</head>` 바로 윗줄에 붙여넣으세요.

### 5단계: 소유권 확인 마무리

이제 다시 아까 열어두었던 **구글 서치 콘솔 화면**으로 돌아오세요.

1.  **[HTML 태그]** 항목이 펼쳐져 있는 상태에서...
2.  아래에 있는 **[확인]** 버튼을 클릭하세요.
3.  초록색 창으로 **"소유권이 확인됨"**이라는 메시지가 뜨면 성공입니다! 🎉
4.  **[속성으로 이동]**을 누르면 이제 구글 서치 콘솔을 사용할 수 있습니다.
