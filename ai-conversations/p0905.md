# 제미나이의 응답

당황하지 마세요! **30초 안에 해결할 수 있습니다.**
지금 바로 아래 순서대로 따라 하세요.

### 🚨 Netlify 긴급 롤백(Rollback) 절차

**1. [Deploys] 탭 클릭**
*   Netlify 사이트 대시보드 상단 메뉴 중 **`Deploys`**를 클릭하세요.

**2. 정상이었던 '이전 버전' 클릭**
*   배포 목록(Deploy log)이 보일 겁니다.
*   맨 위(현재 에러 난 버전) 말고, **그 바로 아래(정상 작동했던 버전)** 항목을 클릭해서 들어가세요.
    *   *팁: 초록색 `Published` 배지가 없는, 이전에 성공했던 항목입니다.*

**3. [Publish this deploy] 버튼 찾기 (핵심!)**
*   상세 페이지로 들어왔나요?
*   **위치:** 화면 상단, 배포 요약 정보(Deploy summary)가 있는 박스 **오른쪽 상단**을 보세요.
*   **`Publish this deploy`** 라는 버튼이 있습니다. (회색 또는 흰색 버튼)
    *   *못 찾겠다면?* `Preview deploy` 버튼 근처를 확인하세요.

**4. 최종 확인**
*   버튼을 누르면 팝업이 뜹니다.
*   **`Publish`** 버튼을 한 번 더 누르세요.
