# 민순의 요청

```text
* 역할: 너는 단일 파일 HTML/CSS/JS 전문 프론트엔드 튜터야.
* 상황: HTML을 전혀 모르는 비개발자 1인 창업가가 `images` 폴더만 겨우 만들어 둔 상태야. 코드가 복잡해질까 봐 겁을 먹고 있어.
* 과업: 
  - 기획안을 바탕으로 `index.html` 전체 코드를 짜 줘.
  - 이미지 경로는 `images` 폴더로 설정해 줘.
* 제약: 
  - 복사해서 붙여넣으면 바로 작동하는 완성된 코드를 제공해 줘.
  - 모바일에서 보기 편한 '반응형'으로 해 줘.
  - 이미지 경로 부분에 `<!-- 이미지 파일명: gift.jpg -->`와 같이 내가 알아볼 수 있는 한글 주석을 반드시 달아 줘.
* 출력: `index.html` 전체 코드
* 입력: [기획안 내용 붙여넣기]
```

# 제미나이의 응답

입력해주신 부분에 구체적인 기획안 내용이 빠져 있어서, **"1인 창업가가 가장 많이 사용하는 표준 랜딩 페이지(상품/서비스 소개)"** 구조로 코드를 작성해 드렸습니다.

이 코드는 **복사해서 메모장에 붙여넣고 `index.html`로 저장**하면 바로 작동합니다.
겁먹지 마세요! 주석(설명글)을 아주 자세히 달아두었으니, 한글로 된 부분만 살짝 고치면 됩니다.

### 📂 준비사항
1. 바탕화면에 폴더를 하나 만듭니다 (예: `my-website`).
2. 그 안에 `images` 라는 이름의 폴더를 만듭니다.
3. `images` 폴더 안에 사용할 사진들을 넣습니다. (아래 코드의 주석을 보고 파일명을 맞춰주세요)

---

### 💻 `index.html` 전체 코드

```html

            max-width: 1000px;
            margin: 0 auto;

        }

            border-bottom: 1px solid #eee;
        }

            background-color: #f9f9f9;
        }

        .hero p { font-size: 18px; color: #666; margin-bottom: 30px; }

            margin: 0 auto;

            box-shadow: 0 10px 20px rgba(0,0,0,0.1);
            border-radius: 12px;

        }

        /* 특징 섹션 (3단 컬럼) */
        .features { padding: 60px 20px; }
        .features h3 { text-align: center; margin-bottom: 40px; font-size: 28px; }
        .feature-grid {
            display: flex;
            flex-wrap: wrap;
            gap: 20px;
            justify-content: center;
        }

            flex: 1 1 300px; /* 모바일에서는 한 줄, PC에서는 3등분 */
            text-align: center;
            padding: 20px;
            border: 1px solid #eee;
            border-radius: 10px;
        }
        .feature-icon {
            width: 80px;
            height: 80px;
            margin: 0 auto 15px;
            object-fit: cover;
            border-radius: 50%; /* 원형 이미지 */
        }

        /* 구매/문의 버튼 */
        .cta-button {
            display: inline-block;
            background-color: #007bff; /* 파란색 버튼 */
            color: white;
            padding: 15px 30px;
            text-decoration: none;
            font-size: 18px;
            font-weight: bold;
            border-radius: 50px;
            transition: background 0.3s;
            margin-top: 20px;
            border: none;
            cursor: pointer;
        }
        .cta-button:hover { background-color: #0056b3; }

        /* 푸터 */
        footer {
            text-align: center;
            padding: 40px 20px;
            background-color: #333;
            color: #aaa;
            font-size: 14px;
        }

        /* 모바일 최적화 */
        @media (max-width: 768px) {
            .hero h2 { font-size: 26px; }
            .hero p { font-size: 16px; }
        }
    </style>
</head>
<body>

    <!-- 1. 상단 로고 영역 -->
    <header>
        <!-- 서비스 이름이나 로고 텍스트를 적으세요 -->
        <h1>내 브랜드 이름</h1>
    </header>

    <!-- 2. 메인 소개 영역 (가장 중요한 부분) -->
    <section class="hero">
        <div class="container">
            <!-- 고객을 사로잡을 메인 문구 -->
            <h2>한 줄로 고객의 마음을 사로잡으세요</h2>
            <p>여기는 부가 설명입니다. 우리 서비스가 왜 좋은지 짧게 설명해주세요.</p>
            
            <!-- 메인 이미지 -->
            <div class="hero-img-box">
                <!-- 아래 src="..." 부분에 images 폴더 안의 파일명을 적으세요 -->
                <img src="images/main-banner.jpg" alt="메인 배너 이미지">
                <!-- 이미지 파일명: main-banner.jpg (대표 이미지 파일명을 여기에 맞춰주세요) -->
            </div>

            <!-- 행동 유도 버튼 -->
            <button class="cta-button" onclick="alert('문의하기 기능은 아직 연결되지 않았습니다!')">
                지금 바로 시작하기
            </button>
        </div>
    </section>

    <!-- 3. 특징/장점 소개 영역 -->
    <section class="features container">
        <h3>왜 우리 서비스를 선택해야 할까요?</h3>
        
        <div class="feature-grid">
            <!-- 특징 1 -->
            <div class="feature-item">
                <img src="images/icon1.jpg" alt="특징1 아이콘" class="feature-icon">
                <!-- 이미지 파일명: icon1.jpg (작은 아이콘이나 사진) -->
                <h4>첫 번째 장점</h4>
                <p>이 장점이 고객에게 어떤 이득을 주는지 설명하세요.</p>
            </div>

            <!-- 특징 2 -->
            <div class="feature-item">
                <img src="images/icon2.jpg" alt="특징2 아이콘" class="feature-icon">
                <!-- 이미지 파일명: icon2.jpg (작은 아이콘이나 사진) -->
                <h4>두 번째 장점</h4>
                <p>경쟁사와 다른 우리만의 차별점을 적어주세요.</p>
            </div>

            <!-- 특징 3 -->
            <div class="feature-item">
                <img src="images/icon3.jpg" alt="특징3 아이콘" class="feature-icon">
                <!-- 이미지 파일명: icon3.jpg (작은 아이콘이나 사진) -->
                <h4>세 번째 장점</h4>

        </div>
    </section>

    <!-- 4. 하단 정보 영역 -->
    <footer>
        <p>&copy; 2024 내 브랜드 이름. All rights reserved.</p>
        <p>문의: contact@mybrand.com</p>
    </footer>

    <script>
        // 자바스크립트 영역: 버튼 클릭 시 간단한 동작
        // 지금은 복잡한 기능 없이 비워두어도 됩니다.
        console.log("홈페이지가 정상적으로 로드되었습니다.");
    </script>

</body>
</html>
```

### 💡 튜터의 팁 (이렇게 수정하세요)

1.  **이미지 파일 준비:**
    *   `images` 폴더 안에 다음 이름으로 사진 4장을 넣어주세요. (확장자가 png라면 코드에서 `.jpg`를 `.png`로만 바꾸면 됩니다)
        *   `main-banner.jpg` (가장 큰 메인 사진)
        *   `icon1.jpg`, `icon2.jpg`, `icon3.jpg` (작은 설명용 사진)
2.  **텍스트 수정:**
    *   코드 중간중간 하얀색 글씨로 된 한글 부분(예: `한 줄로 고객의 마음을 사로잡으세요`)을 원하시는 내용으로 바꿔 적으세요.
3.  **확인:**
    *   저장 후 `index.html` 파일을 더블 클릭하면 크롬/엣지 브라우저에서 바로 뜹니다.
    *   창 크기를 줄였다 늘렸다 해보세요. 모바일에서도 잘 보이도록 자동으로 조절됩니다!
