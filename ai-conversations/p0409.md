# 민순의 요청

```text
* 역할: 너는 제미나이 캔버스 사용에 능숙한 프론트엔드 개발 튜터야.
* 상황: 이미지를 폴더에 넣어두고 코딩을 시도하고 있어.
* 과업: 
  1. 기획안을 바탕으로 HTML, CSS, JS가 합쳐진 'index.html' 파일용 전체 코드를 짜 줘.
  2. 'images' 폴더에 있는 사진들을 가져오도록 코드를 작성하고, 이미지 경로 부분에 <!-- 이미지 파일명: gift.jpg -->와 같이 한글 주석을 반드시 달아 줘.
* 제약: 
  - 코드를 복사해서 메모장에 붙여넣으면 바로 작동하게 해 줘.
  - 디자인은 모바일에서 보기에 깔끔한 '반응형'으로 해 줘.
* 출력: 단일 HTML 코드 블록
* 입력: [디자인 기획안 및 스타일 정보]
 (예시: 주조색 #FF7F50, 배경 #F5F5DC, 폰트 Noto Sans KR, 위에서 만든 화면 설계도 내용. 이미지는 'perfecta-mvp' 폴더 안의 'images' 폴더에 있어.)
```

# 제미나이의 응답

네, 알겠습니다! 제미나이 캔버스 튜터로서, 요청하신 디자인 기획안(주조색 #FF7F50, 배경 #F5F5DC, 폰트 Noto Sans KR)을 반영하여 **모바일 퍼스트 반응형 웹페이지** 코드를 작성해 드릴게요.

이 코드는 `index.html` 파일 하나에 모든 내용이 들어있습니다.
**`perfecta-mvp` 폴더 안에 이 파일을 저장**하고, 같은 폴더 내에 **`images` 폴더**가 있어야 이미지가 정상적으로 보입니다.
