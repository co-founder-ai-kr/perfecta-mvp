# 민순의 요청

```text
* 역할: 너는 구글 시트와 JSON 데이터 구조 전문가야.
* 상황: 구글 시트에 정리된 데이터를 웹 사이트 개발에 바로 쓸 수 있는 JSON 포맷으로 변환하고 싶어.
* 과업: 
  1. 아래 데이터를 올바른 JSON 형식(키 값: product_name, price, description, image_url)으로 변환하고 문법 오류를 검사해 줘.
  2. 구글 폼을 내 웹 사이트에 삽입하는 코드를 알려 줘.
* 제약: 변환된 JSON 코드는 복사해서 바로 사용할 수 있는 코드 블록 형태로 출력해 줘.
* 출력: 
  1. JSON 코드 블록
  2. HTML 삽입 코드
* 입력: [데이터 붙여넣기]
 (예시: product_name, price, description, image_url\n텀블러, 25000, "환경을 생각하는 선물", "https://example.com/tumbler.jpg"\n무드등, 35000, "감성적인 분위기 연출", "https://example.com/moodlamp.jpg")
```

# 제미나이의 응답

네, 구글 시트와 JSON 데이터 구조 전문가입니다. 요청하신 데이터를 웹 개발에 즉시 사용할 수 있는 JSON 포맷으로 변환하고, 구글 폼 삽입 코드를 작성해 드리겠습니다.

### 1. JSON 코드 블록

제공해주신 예시 데이터를 기반으로 변환한 결과입니다.
*   **검사 완료:** 문법 오류(Syntax Error)를 확인했으며, `price`는 숫자형(Number)으로, 나머지는 문자열(String)로 처리하여 활용도를 높였습니다.

```json
[
  {
    "product_name": "텀블러",
    "price": 25000,
    "description": "환경을 생각하는 선물",
    "image_url": "https://example.com/tumbler.jpg"
  },
  {
    "product_name": "무드등",
    "price": 35000,
    "description": "감성적인 분위기 연출",
    "image_url": "https://example.com/moodlamp.jpg"
  }
]
```

---

### 2. 구글 폼 HTML 삽입 코드

웹 사이트에 구글 폼을 넣기 위해서는 `<iframe>` 태그를 사용합니다. 아래 코드를 복사하여 웹 페이지의 HTML 파일 원하는 위치에 붙여넣으세요.

**사용법:**
1.  아래 코드의 `src="..."` 부분에 있는 주소를 **본인의 구글 폼 주소**로 변경해야 합니다.
    *   *(구글 폼 화면 우측 상단 '보내기' 버튼 클릭 -> `< >` 아이콘 클릭 -> HTML 복사)*
2.  `width`와 `height` 값을 조절하여 사이트에 맞는 크기로 변경할 수 있습니다.

```html
<!-- 구글 폼 임베드 시작 -->
<div style="text-align: center;">
  <iframe 
    src="https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?embedded=true" 
    width="640" 
    height="800" 
    frameborder="0" 
    marginheight="0" 
    marginwidth="0">
    로드 중...
  </iframe>
</div>
<!-- 구글 폼 임베드 끝 -->
```
