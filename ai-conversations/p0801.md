# 민순의 요청

```text
* 역할: 너는 구글 시트와 JSON 데이터 구조 전문가야.
* 상황: 구글 시트에 정리된 데이터를 웹 사이트 개발에 바로 쓸 수 있는 JSON 포맷으로 변환하고 싶어.
* 과업: 
  1. 아래 데이터를 올바른 JSON 형식(키 값: product_name, price, description, image_url)으로 변환하고 문법 오류를 검사해 줘.
  2. 구글 폼을 내 웹 사이트에 삽입하는 코드를 알려 줘.
* 제약: 변환된 JSON 코드는 복사해서 바로 사용할 수 있는 코드 블록 형태로 출력해 줘.
* 출력: 
  1. JSON 코드 블록
  2. HTML 삽입 코드
* 입력: [데이터 붙여넣기]
 (예시: 
  상품명    가격    설명    이미지 주소
  텀블러    25000   환경을 생각하는 선물    https://example.com/tumbler.jpg
  무드등    35000   감성적인 분위기 연출    https://example.com/moodlamp.jpg)
```

# 제미나이의 응답

제시해주신 예시 데이터를 바탕으로 요청하신 작업을 수행했습니다.

### 1. JSON 코드 블록
입력해주신 예시 데이터(텀블러, 무드등)를 지정된 키 값(`product_name`, `price`, `description`, `image_url`)에 맞춰 변환했습니다. 가격(price)은 숫자형으로 처리하여 계산이 용이하도록 했으며, 문법 오류 검사를 마쳤습니다.

```json
[
  {
    "product_name": "텀블러",
    "price": 25000,
    "description": "환경을 생각하는 선물",
    "image_url": "https://example.com/tumbler.jpg"
  },
  {
    "product_name": "무드등",
    "price": 35000,
    "description": "감성적인 분위기 연출",
    "image_url": "https://example.com/moodlamp.jpg"
  }
]
```

---

### 2. 구글 폼 HTML 삽입 코드
구글 폼을 웹사이트에 넣으려면 `<iframe>` 태그를 사용해야 합니다. 아래 코드를 복사하여 웹사이트의 HTML 파일 원하는 위치에 붙여넣으세요.

**사용 방법:**
1. 구글 폼 편집 화면 우측 상단의 **[보내기]** 버튼 클릭
2. **'< >' (HTML 삽입)** 아이콘 클릭
3. 표시된 HTML 코드를 복사하거나, 아래 코드의 `src="..."` 부분에 있는 주소만 본인의 폼 주소로 변경해서 사용하세요.

```html
<!-- 구글 폼 임베드 코드 시작 -->
<iframe 
  src="https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?embedded=true" 
  width="640" 
  height="800" 
  frameborder="0" 
  marginheight="0" 
  marginwidth="0">
  로드 중...
</iframe>
<!-- 구글 폼 임베드 코드 끝 -->
```

*참고: `width`(너비)와 `height`(높이) 값은 웹사이트 레이아웃에 맞춰 숫자를 조정하시면 됩니다.*
