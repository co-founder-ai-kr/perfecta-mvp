# 민순의 요청

```text
* 역할: 너는 구글 시트와 JSON 데이터 구조 전문가야.
* 상황: 비개발자 1인 창업가가 구글 시트의 선물 목록을 웹 사이트에서 사용하기 위해 JSON으로 변환하려 해.
* 과업: 
  1. 아래 데이터를 올바른 JSON 형식(키 값: product_name, price, description, image_url)으로 변환하고 문법 오류를 검사해 줘.
  2. 구글 폼을 내 웹 사이트에 삽입하는 코드를 알려 줘.
* 제약: 변환된 JSON 코드는 복사해서 바로 사용할 수 있는 코드 블록 형태로 출력해 줘.
* 출력: 
  1. JSON 코드 블록
  2. HTML 삽입 코드
* 입력: [데이터 붙여넣기]
 (예시: product_name, price, description, image_url\n텀블러, 25000, "환경을 생각하는 선물", "https://example.com/tumbler.jpg"\n무드등, 35000, "감성적인 분위기 연출", "https://example.com/moodlamp.jpg")
```

# 제미나이의 응답

제공해주신 예시 데이터를 바탕으로 변환한 **JSON 코드**와 웹 사이트에 구글 폼을 넣을 수 있는 **HTML 삽입 코드**입니다.

### 1. JSON 코드 블록
입력하신 예시 데이터(텀블러, 무드등)를 표준 JSON 형식으로 변환했습니다.
*   **문법 검사 완료:** 쉼표(,), 따옴표("), 괄호([, {) 등의 문법 오류를 모두 확인했습니다.
*   **데이터 타입:** `price`는 숫자형(Number)으로, 나머지는 문자열(String)로 처리하여 개발 활용도를 높였습니다.

```json
[
  {
    "product_name": "텀블러",
    "price": 25000,
    "description": "환경을 생각하는 선물",
    "image_url": "https://example.com/tumbler.jpg"
  },
  {
    "product_name": "무드등",
    "price": 35000,
    "description": "감성적인 분위기 연출",
    "image_url": "https://example.com/moodlamp.jpg"
  }
]
```

### 2. 구글 폼 HTML 삽입 코드 (Iframe)
웹 사이트의 원하는 위치에 아래 코드를 복사해서 붙여넣으세요.

**사용 방법:**
1.  아래 코드에서 `src="https://docs.google.com/forms/..."` 부분의 주소를 본인의 구글 폼 주소로 변경해야 합니다.
2.  **구글 폼 주소 가져오는 법:** 구글 폼 편집 화면 우측 상단 **[보내기]** 버튼 클릭 -> **[< >] (임베드)** 아이콘 클릭 -> **HTML 복사**

```html
<!-- 구글 폼 임베드 시작 -->
<div style="width: 100%; display: flex; justify-content: center;">
  <iframe 
    src="https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?embedded=true" 
    width="640" 
    height="800" 
    frameborder="0" 
    marginheight="0" 
    marginwidth="0"
    title="선물 신청 구글 폼">
    로드 중...
  </iframe>
</div>
<!-- 구글 폼 임베드 끝 -->
```

**팁:** `width="100%"`로 설정하면 모바일에서도 화면 너비에 맞춰 자동으로 조절됩니다. 높이(`height`)는 폼의 길이에 맞춰 숫자를 조절해 주세요.
