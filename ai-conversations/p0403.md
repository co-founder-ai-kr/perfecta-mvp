# 민순의 요청

```text
* 역할: 너는 고객 중심적 사고를 가진 마케터야.
* 상황: 개발자가 쓴 딱딱한 기능 명세서를 고객이 혹할 만한 혜택 중심의 언어로 바꾸고 싶어.
* 과업: 이 기능들을 고객이 얻을 수 있는 혜택이랑 감정적인 가치 위주로 싹 바꿔서 써 줘.
* 제약:
  - 어려운 기술 용어는 다 빼고, 고객이 읽자마자 '아, 이건 내 이야기다!'라고 느끼게 해 줘.
  - 어떤 기술을 어떤 혜택으로 바꿨는지 주석으로 달아 줘.
* 입력: [기능 명세서 내용]
 (예시: AI 관계 분석 알고리즘, 메시지 카드 자동 생성 기능)
```

# 제미나이의 응답

네, 알겠습니다! 개발자분들이 작성하신 기능 명세서는 '무엇을 만들었는지(What)'에 집중되어 있지만, 고객은 '이게 나한테 왜 좋은지(Why)'를 알고 싶어 하죠.

입력해주신 예시(AI 관계 분석, 메시지 카드 생성)를 포함하여, **'인맥 관리 및 선물하기 서비스'**라고 가정하고 몇 가지 기능을 추가해 매력적인 마케팅 언어로 바꿔보겠습니다.

---

### 💌 [고객을 위한 혜택 명세서]

개발자의 언어를 고객의 언어로 번역했습니다.

#### 1. "소중한 사람과 멀어지지 않도록, 타이밍을 놓치지 않게 해드려요."
*   **원래 기능 명세:** AI 관계 분석 알고리즘 (연락 빈도 및 텍스트 분석을 통한 친밀도 스코어링)
*   **💡 마케터의 주석 (Tech → Benefit):**
    *   '알고리즘', '스코어링' 같은 차가운 단어는 뺐습니다.
    *   고객의 진짜 두려움은 '친한 친구와 소원해지는 것'입니다.
    *   **핵심 가치:** 관계 유지에 대한 **안심감**과 **적절한 타이밍** 제공.

#### 2. "고민은 짧게, 감동은 길게. 진심을 담은 문장을 대신 써드릴게요."
*   **원래 기능 명세:** 메시지 카드 자동 생성 기능 (상황별 키워드 기반 텍스트 제너레이션)
*   **💡 마케터의 주석 (Tech → Benefit):**
    *   '자동 생성'이라는 말은 기계적인 느낌을 줍니다. 대신 '고민을 덜어준다'는 혜택을 강조했습니다.
    *   축하 인사를 보낼 때 "뭐라고 쓰지?" 고민하는 고통(Pain point)을 해결해 줍니다.
    *   **핵심 가치:** 작문의 **부담감 해소**와 상대방에게 전할 **감동**.

#### 3. "센스 있는 친구라는 말, 듣게 해드릴게요."
*   **원래 기능 명세:** 협업 필터링 기반 선물 추천 엔진 (유사 사용자 그룹 구매 이력 분석)
*   **💡 마케터의 주석 (Tech → Benefit):**
    *   '협업 필터링'은 고객이 알 필요가 없습니다.
    *   선물을 고를 때 가장 원하는 결과는 상대방의 만족과 나의 평판(센스 있다)입니다.
    *   **핵심 가치:** 선물 선택 실패에 대한 **두려움 제거** 및 **자존감 상승**.

#### 4. "가장 여유로운 시간에, 똑똑하게 알려드려요."
*   **원래 기능 명세:** 사용자 행동 패턴 기반 푸시 알림 스케줄링 (앱 사용 시간대 분석 및 발송)
*   **💡 마케터의 주석 (Tech → Benefit):**
    *   '스케줄링', '패턴 분석' 대신 '여유로운 시간'이라는 고객의 상황을 묘사했습니다.
    *   바쁠 때 울리는 알림은 소음이지만, 한가할 때 울리는 알림은 정보가 됩니다.
    *   **핵심 가치:** 방해받지 않을 **권리 존중** 및 **편의성**.

#### 5. "우리의 추억, 휴대폰을 잃어버려도 절대 사라지지 않아요."
*   **원래 기능 명세:** 클라우드 자동 동기화 및 이중 백업 시스템
*   **💡 마케터의 주석 (Tech → Benefit):**
    *   '클라우드', '백업'은 기능이지만, 고객에게는 '추억 보존'입니다.
    *   기술적 안정성을 감성적인 영속성으로 치환했습니다.
    *   **핵심 가치:** 소중한 데이터 유실에 대한 **불안감 해소**.

---

**마케터의 한마디:**
보시다시피 기술 용어를 **'고객이 얻게 될 감정(안심, 뿌듯함, 편안함)'**과 **'해결되는 문제(고민 시간 단축, 관계 단절 예방)'**로 바꾸면 훨씬 더 매력적으로 다가옵니다. 추가로 변환하고 싶은 기능이 있다면 언제든 말씀해 주세요!
