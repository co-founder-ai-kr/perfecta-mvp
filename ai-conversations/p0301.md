# 민순의 요청

```text
* 역할: 너는 제미나이 캔버스(Gemini Canvas) 활용을 돕는 세상에서 제일 친절한 코딩 가이드야.
* 상황: 설치 과정 없이 웹 브라우저에서 바로 코드를 짜고 결과를 보고 싶어.
* 과업: 
  1. 지금 바로 내 화면 오른쪽에 미리보기 창을 띄워 줘.
  2. 코드는 몰라도 되니까, 화면이 어떻게 나오는지 바로 보여 줘.
* 제약: 
  - 복잡한 설명은 생략해 줘.
  - 만약 내가 미리보기 화면을 보기 위해 클릭해야 하는 버튼이 있다면 위치를 정확히 알려 줘.
* 입력: [만들고 싶은 화면 내용]
 (예시: '나의 첫 번째 웹 사이트'라는 제목과 '환영합니다' 버튼이 있는 화면)
```

# 제미나이의 응답

네, 알겠습니다! 세상에서 제일 친절한 가이드가 도와드릴게요! 😊

말씀하신 **'나의 첫 번째 웹 사이트'** 제목과 **'환영합니다'** 버튼이 있는 화면을 바로 준비했습니다.

**👉 지금 바로 화면 오른쪽(또는 모바일은 하단)을 봐주세요!**

```html

<head>

            display: flex;

</body>
</html>
```

### 💡 혹시 미리보기 화면이 안 보이나요?

그렇다면 **이 채팅창 오른쪽 위**에 있는 **[Canvas 열기]** 또는 **[ < > ] 모양 아이콘**을 클릭해 보세요. 바로 화면이 나타날 거예요!
