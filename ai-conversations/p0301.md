# 제미나이의 응답

안녕하세요! 제미나이 앱을 자주 다루는 웹 개발자 동료로서, **제미나이 캔버스(Gemini Canvas)**의 세계로 안내해 드릴게요.

캔버스는 단순히 채팅만 하는 게 아니라, **코딩이나 글쓰기 같은 복잡한 작업을 별도의 창에서 시각적으로 확인하며 수정할 수 있는 작업 공간**입니다. 개발자 입장에서는 일종의 '실시간 에디터'나 'IDE(통합 개발 환경)'처럼 느껴져서 아주 유용하죠.

공유해주신 구글 도움말 문서를 바탕으로, **활성화 방법**과 **미리보기 방법**을 아주 쉽게 단계별로 알려드릴게요.

---

### 1. 캔버스 활성화하기 (시작하는 법)

캔버스는 제미나이(gemini.google.com)에 접속해서 바로 사용할 수 있는데, 크게 두 가지 방법으로 열립니다.

**방법 A: 자연스럽게 열기 (추천)**
제미나이가 "아, 이건 캔버스가 필요하겠구나"라고 판단하면 자동으로 열립니다. 주로 **코딩**이나 **긴 글쓰기**를 요청할 때 그렇습니다.

1.  프롬프트 입력창에 다음과 같이 입력해 보세요.
    *   **코딩 예시:** "간단한 투두 리스트(To-do List) 웹페이지를 HTML, CSS, JS로 만들어 줘."
    *   **글쓰기 예시:** "생성형 AI의 미래에 대한 블로그 글을 작성해 줘."
2.  제미나이가 답변을 생성하면서 화면이 **좌우로 분할**되고, 오른쪽에 캔버스 창이 자동으로 뜹니다.

**방법 B: 수동으로 열기**
이미 대화 중이거나, 강제로 캔버스를 열고 싶을 때 씁니다.

1.  프롬프트에 **"캔버스 열어줘"** 또는 **"Open canvas"**라고 입력합니다.
2.  혹은, 답변 내용 위에 마우스를 올렸을 때 나타나는 **'캔버스에서 열기(Open in Canvas)' 아이콘** (보통 문서 모양 아이콘)을 클릭합니다.

> **💡 개발자 팁:** 캔버스는 현재 **Gemini 1.5 Pro** 모델에서 가장 잘 작동합니다. 혹시 안 열린다면 모델 설정이 Pro 버전인지 확인해 보세요.

---

### 2. 캔버스 미리보기 (결과물 확인하기)

캔버스가 열리면 화면이 반으로 나뉩니다.
*   **왼쪽:** 제미나이와의 채팅창 (수정 요청하는 곳)
*   **오른쪽:** 캔버스 (실제 코드나 글이 작성되는 곳)

여기서 **'미리보기'**는 작업 유형에 따라 다르게 동작합니다.

**CASE 1: 코딩 작업 (웹 개발)**
웹 개발자로서 가장 매력적인 부분입니다. HTML/CSS/JS 코드를 요청했다면:

1.  오른쪽 캔버스 상단에 **[코드(Code)]** 탭과 **[미리보기(Preview)]** 탭이 생길 수 있습니다. (혹은 코드가 바로 렌더링되어 보입니다.)
2.  코드를 수정하면 결과 화면이 실시간으로 반영되는 것을 볼 수 있습니다. 마치 코드펜(CodePen)이나 JSFiddle을 쓰는 것과 비슷합니다.

**CASE 2: 글쓰기 작업**
1.  오른쪽 캔버스에 작성된 글이 깔끔한 문서 형태로 보입니다.
2.  여기서 직접 텍스트를 클릭해서 **타이핑하여 수정**할 수도 있고, 드래그해서 특정 부분만 제미나이에게 **"이 부분 좀 더 부드럽게 고쳐줘"**라고 요청할 수도 있습니다.

### 3. 요약: 딱 이것만 기억하세요!

1.  **접속:** gemini.google.com
2.  **입력:** "HTML로 자기소개 페이지 만들어 줘" (자동 활성화)
