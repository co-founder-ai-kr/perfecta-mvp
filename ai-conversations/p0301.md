# 민순의 요청

```text
* 역할: 너는 제미나이 앱(gemini.google.com)을 잘 아는 웹 개발자야.
* 상황: 제미나이 캔버스가 어떻게 동작하는지 궁금해.
* 과업:
  1. 캔버스 활성화하는 방법 알려 줘.
  2. 캔버스 창에서 HTML 짠 것 미리보기 하는 방법 알려 줘.
* 제약: 최신 화면 기준으로 설명해 줘.
```

# 제미나이의 응답

반갑습니다! 제미나이(Gemini) 앱을 자주 사용하는 웹 개발자로서, **제미나이 캔버스(Gemini Canvas)** 기능을 최신 인터페이스 기준으로 명쾌하게 설명해 드릴게요.

제미나이 캔버스는 챗GPT의 'Canvas'나 Claude의 'Artifacts'와 유사하게, **채팅창과 분리된 별도의 작업 공간(오른쪽 창)**을 열어 코드를 작성하거나 긴 글을 편집할 수 있는 기능입니다.

요청하신 두 가지 과업에 대해 단계별로 알려드립니다.

---

### 1. 캔버스 활성화하는 방법

제미나이 캔버스는 별도의 '설정' 메뉴에서 켜는 것이 아니라, **프롬프트(명령어)의 의도에 따라 자동으로 활성화**되거나, 생성된 답변에서 수동으로 열 수 있습니다.

**방법 A: 프롬프트로 자동 활성화 (가장 확실한 방법)**
제미나이에게 **"코드를 짜줘"** 혹은 **"긴 글을 써줘"**와 같이 창작이나 수정이 필요한 요청을 하면 자동으로 캔버스가 열립니다.

1.  제미나이 채팅창에 다음과 같이 입력하세요.
    *   예시: *"HTML과 CSS로 모던한 스타일의 로그인 페이지를 만들어 줘."*
    *   예시: *"파이썬으로 뱀 게임 코드를 짜 줘."*
2.  제미나이가 코드를 생성하면서, 화면이 **좌측(채팅)**과 **우측(캔버스)**으로 분할되며 캔버스 모드가 자동으로 켜집니다.

**방법 B: 수동으로 열기**
만약 제미나이가 채팅창 안에만 코드를 뱉어냈다면, 수동으로 캔버스로 옮길 수 있습니다.

1.  제미나이의 답변(코드 블록이나 긴 텍스트) 위에 마우스를 올립니다.
2.  답변 하단이나 우측 상단에 있는 **[캔버스에서 열기]** (문서 모양 아이콘 혹은 'Open in Canvas') 버튼을 클릭합니다.
3.  해당 내용이 우측 캔버스 창으로 이동합니다.

---

### 2. 캔버스 창에서 HTML 미리보기(Preview) 하는 방법

캔버스가 활성화되어 HTML 코드가 작성되었다면, 브라우저를 따로 켜지 않고도 바로 결과를 확인할 수 있습니다.

1.  **캔버스 창 확인:** 화면 우측에 코드가 적힌 캔버스 창이 떠 있는지 확인합니다.
2.  **보기 모드 전환 (우측 상단):**
    *   캔버스 창의 우측 상단을 보면 보통 두 가지 탭이나 아이콘이 있습니다.
    *   **코드(Code):** HTML/CSS/JS 소스 코드를 보고 수정하는 화면.
    *   **미리보기(Preview) / 실행(Run):** 코드가 렌더링 된 실제 웹페이지 화면.
3.  **미리보기 클릭:** **[미리보기]** 탭을 클릭하면 즉시 렌더링 된 웹페이지가 나타납니다.
    *   *참고:* HTML/CSS/JS가 포함된 웹 코드의 경우, 제미나이가 코드를 다 작성하자마자 **자동으로 미리보기 탭을 먼저 보여주는 경우**가 많습니다.

**💡 개발자 팁:**
*   **실시간 수정:** '코드' 탭에서 `<h1>` 태그의 내용을 바꾸거나 CSS 색상을 변경하면, '미리보기' 탭에 즉시 반영되거나 [실행] 버튼을 눌러 바로 확인할 수 있습니다.
*   **부분 수정 요청:** 코드를 직접 건드리지 않고, 채팅창(좌측)에서 *"버튼 색깔을 파란색으로 바꿔줘"*라고 말하면 캔버스(우측)의 코드가 자동으로 수정되고 미리보기도 업데이트됩니다.

이 기능을 활용하면 VS Code 같은 에디터를 켜지 않고도 간단한 UI 프로토타이핑을 아주 빠르게 끝낼 수 있습니다!
