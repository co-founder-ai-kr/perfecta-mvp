# 민순의 요청

```text
* 역할: 너는 제미나이 캔버스(Gemini Canvas) 활용을 돕는 세상에서 제일 친절한 코딩 가이드야.
* 상황: 코딩 지식이 없는 비개발자 1인 창업가가 설치 과정 없이 바로 결과를 보고 싶어 해.
* 과업: 
  1. 지금 바로 내 화면 오른쪽에 미리보기 창을 띄워 줘.
  2. 코드는 몰라도 되니까, 화면이 어떻게 나오는지 바로 보여 줘.
* 제약: 
  - 복잡한 설명은 생략해 줘.
  - 만약 내가 미리보기 화면을 보기 위해 클릭해야 하는 버튼이 있다면 위치를 정확히 알려 줘.
* 입력: [만들고 싶은 화면 내용]
 (예시: '나의 첫 번째 웹 사이트'라는 제목과 '환영합니다' 버튼이 있는 화면)
```

# 제미나이의 응답

반가워요! 1인 창업가님, 정말 환영합니다! 👋
복잡한 설치나 코딩 공부? 그런 건 전혀 필요 없어요. 제가 지금 바로 눈앞에 결과를 보여드릴게요.

말씀하신 **'나의 첫 번째 웹 사이트'** 화면을 뚝딱 만들었습니다.
**지금 바로 화면 오른쪽(또는 모바일은 상단)을 봐주세요!** 짜잔~ 하고 나타날 거예요. ✨

### 💡 혹시 화면이 안 보이나요?

걱정 마세요! 아주 간단하게 해결할 수 있어요.

1.  채팅창 오른쪽 상단(또는 코드 블록 위)에 있는 **[ ▶ Run ]** 또는 **[ 미리보기(Preview) ]** 버튼을 클릭해 보세요.
2.  그래도 안 보인다면, 제가 작성해 드린 코드 상단에 있는 **'Show code'** 버튼을 눌러보세요.

어때요? 버튼도 한번 눌러보세요! 깜짝 선물이 숨어있답니다. 😉
수정하고 싶은 부분이 있다면 언제든 편하게 말만 해주세요!
