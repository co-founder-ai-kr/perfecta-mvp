# 민순의 요청

```text
* 역할: 너는 구글 AI 스튜디오의 친절한 가이드야.
* 상황: 나는 코딩을 전혀 모르는 1인 창업가인데, API를 이용해 자동화된 직원을 만들고 싶어서 구글 AI 스튜디오에 처음 접속했어. 메뉴가 너무 많아 어디를 눌러야 할지 막막해.
* 과업: 
  1. 'System Instructions' 메뉴가 현재 화면의 어디에 있는지 정확한 위치를 알려 줘.
  2. 그곳에 입력할 시스템 프롬프트 예시를 작성해 줘.
* 제약: 
  - 메뉴 위치는 "왼쪽 상단 톱니 바퀴 아이콘"처럼 초등학생도 찾을 수 있게 묘사해 줘.
  - 여기서 설정한 내용이 API 호출 시 어떤 역할을 하는지 한 줄로 쉽게 설명해 줘.
* 출력: 
  1. 위치 설명 (글머리 기호)
  2. 프롬프트 예시 (코드 블록)
* 입력: [에이전트 역할]
 (예시: 선물 큐레이터)
```

# 제미나이의 응답

반가워요! 코딩을 몰라도 괜찮아요. 구글 AI 스튜디오라는 멋진 도구를 찾아오신 것만으로도 이미 절반은 성공하신 거예요! 🥳

복잡해 보이는 화면 때문에 놀라셨죠? 제가 **'선물 큐레이터'**라는 직원을 만든다고 가정하고, 아주 쉽게 딱 필요한 곳만 알려드릴게요.

### 1. 위치 설명

*   **어디를 봐야 하나요?**
    *   화면 **왼쪽 맨 위**를 봐주세요.
    *   **'System Instructions'**라고 적힌 **네모난 입력 상자**가 보일 거예요. (채팅창 바로 위에 있어요!)
    *   만약 안 보인다면, 왼쪽 위에 있는 **'System Instructions'라는 글자 옆의 작은 화살표(∨)**를 누르면 상자가 짠! 하고 나타나요.

*   **이게 무슨 역할을 하나요?**
    *   👉 **"AI에게 '너는 지금부터 선물 추천 전문가야!'라고 최면을 걸어, 앞으로의 모든 대답을 그 역할에 맞춰서 하게 만드는 설정이에요."**

---

### 2. 프롬프트 예시 (선물 큐레이터)

아래 내용을 복사해서 방금 찾은 **'System Instructions'** 상자에 붙여넣기만 하면 돼요.

```markdown
너는 지금부터 센스 넘치는 '전문 선물 큐레이터'야. 
너의 목표는 고객이 누구에게 선물을 줄지(대상), 예산은 얼마인지, 어떤 상황인지(생일, 기념일 등)를 듣고 가장 완벽한 선물을 추천해 주는 거야.

[행동 지침]
1. 말투: 백화점 VIP 담당 매니저처럼 매우 정중하고 친절하게, 하지만 너무 딱딱하지 않게 대화해 줘. (이모지 적절히 사용)
2. 질문: 고객이 정보를 부족하게 주면, 더 좋은 추천을 위해 자연스럽게 추가 질문을 해 줘. (예: "받으시는 분의 연령대가 어떻게 되시나요?")
3. 추천 방식: 선물을 추천할 때는 반드시 다음 3가지를 포함해서 3가지 옵션을 제안해 줘.
   - 상품명
   - 대략적인 가격대
   - 이 선물을 추천하는 감성적인 이유
4. 마무리: 고객이 선택을 고민하면 격려해 주고, 좋은 선물이 될 거라고 확신을 줘.

자, 이제 고객이 들어왔다고 생각하고 준비해!
```

이제 이 내용을 입력하고 오른쪽 채팅창에 "여자친구 1주년 선물 추천해 줘"라고 말해보세요. AI가 완벽한 직원처럼 대답할 거예요! 또 궁금한 게 있으면 언제든 물어보세요. 화이팅! 🚀
