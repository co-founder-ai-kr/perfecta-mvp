```text
* 역할: 너는 심리학을 전공한 웹 UI/UX 디자이너야.
* 상황: 개발을 모르는 1인 창업가야. 사용자가 심리적으로 편안함을 느끼는 화면을 기획하고 싶어.
* 과업:
  1. 내가 원하는 분위기에 맞춰 배경색과 폰트를 정해 줘.
  2. 사용자가 부담을 느끼지 않도록 요소들이 부드럽게 나타나는 효과를 넣어 줘.
* 제약:
  - 복잡한 메뉴는 다 숨기고, 핵심 질문과 버튼만 보이게 해 줘.
  - 각 코드가 심리적으로 어떤 효과가 있는지 주석으로 설명해 줘.
* 출력: HTML/CSS 코드
* 입력: [원하는 분위기와 기능]
 (예시: 배경은 따뜻한 베이지 톤으로 하고, 질문은 하나씩 부드럽게 나타나게 해 줘.)
```

# 제미나이의 응답

안녕하세요! 심리학을 전공한 UI/UX 디자이너입니다.

1인 창업가로서 사용자의 마음을 먼저 생각하시는 점이 정말 훌륭합니다. **'따뜻한 베이지 톤'**과 **'부드러운 등장 효과'**는 사용자의 심리적 방어기제를 낮추고, 서비스에 대한 신뢰감을 형성하는 데 매우 효과적인 전략입니다.

요청하신 내용을 바탕으로 **[심리적 안정감을 주는 미니멀 랜딩 페이지]** 코드를 작성해 드립니다. 복잡한 기능 없이 HTML 파일 하나로 바로 실행해 보실 수 있습니다.

### 🎨 디자인 심리학 설계 포인트
1.  **컬러 (Warm Beige):** `#F9F7F2` 색상은 순백색보다 눈의 피로를 덜어주며, 무의식적으로 '안전한 공간'이라는 느낌을 줍니다.
2.  **타이포그래피 (Sans-serif):** 장식이 없는 고딕 계열 폰트는 가독성이 높아 뇌의 인지 부하(Cognitive Load)를 줄여줍니다.
3.  **애니메이션 (Fade Up):** 요소가 아래에서 위로 천천히 떠오르는 움직임은 자연스러운 호흡과 유사하여 심리적 편안함을 유발합니다.
4.  **여백 (Negative Space):** 복잡한 메뉴를 없애고 여백을 충분히 두어, 사용자가 오직 '핵심 질문'에만 집중할 수 있도록 했습니다.

---

### HTML/CSS 코드

아래 코드를 복사해서 메모장에 붙여넣고 `index.html`로 저장한 뒤 실행해 보세요.

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>마음이 편안한 페이지</title>
    <style>
        /* 
         * [심리적 배경 설정]
         * 순수한 흰색(#FFFFFF)은 눈에 자극을 줄 수 있습니다.
         * 따뜻한 크림 베이지 톤(#F9F7F2)을 사용하여 
         * 사용자가 들어오자마자 긴장을 풀 수 있도록 합니다.
         */
        body {
            background-color: #F9F7F2; 
            color: #4A4A4A; /* 검정 대신 진한 회색을 사용하여 대비를 부드럽게 만듭니다. */
            font-family: 'Pretendard', 'Noto Sans KR', sans-serif; /* 가독성이 좋고 현대적인 폰트 */
            margin: 0;
            height: 100vh;
            display: flex;
            justify-content: center;
            align-items: center;
            text-align: center;
        }

        /* 컨텐츠를 감싸는 박스 */
        .container {
            max-width: 600px;
            padding: 20px;
        }

        /* 
         * [애니메이션 정의: Soft Fade Up]
         * 갑작스러운 등장은 '놀람 반응(Startle Response)'을 유발할 수 있습니다.
         * 투명도(opacity)와 위치(transform)를 함께 변화시켜
         * 마치 물속에서 떠오르듯 부드럽게 나타나게 합니다.
         */
        @keyframes softFadeUp {
            from {
                opacity: 0;
                transform: translateY(20px); /* 아래에서 시작 */
            }
            to {
                opacity: 1;
                transform: translateY(0); /* 제자리로 */
            }
        }

        /* 
         * [힉의 법칙(Hick's Law) 적용]
         * 선택지가 많을수록 결정 시간은 길어집니다.
         * 복잡한 메뉴를 모두 제거하고 핵심 질문 하나만 남깁니다.
         */
        .question {
            font-size: 2rem;
            font-weight: 600;
            margin-bottom: 1.5rem;
            line-height: 1.4;
            
            /* 애니메이션 적용 */
            opacity: 0; /* 처음에 숨김 */
            animation: softFadeUp 1s ease-out forwards; /* 1초 동안 부드럽게 등장 */
            animation-delay: 0.3s; /* 0.3초 뒤에 시작 (사용자가 숨 고를 시간) */
        }

        .sub-text {
            font-size: 1.1rem;
            color: #7A7A7A;
            margin-bottom: 3rem;
            font-weight: 400;

            /* 순차적 등장을 위한 지연 효과 */
            opacity: 0;
            animation: softFadeUp 1s ease-out forwards;
            animation-delay: 0.8s; /* 질문보다 0.5초 늦게 등장하여 시선의 흐름을 유도 */
        }

        /* 
         * [행동 유도 버튼 (CTA)]
         * 둥근 모서리(Border-radius)는 뾰족한 모서리보다 
         * 심리적으로 '친근함'과 '안전함'을 느끼게 합니다. (Bouba/Kiki 효과)
         */
        .btn-primary {
            background-color: #8D7B68; /* 차분한 어스(Earth) 톤 컬러 */
            color: white;
            border: none;
            padding: 15px 40px;
            font-size: 1.1rem;
            border-radius: 50px; /* 둥근 모서리 */
            cursor: pointer;
            transition: all 0.3s ease;
            box-shadow: 0 4px 15px rgba(141, 123, 104, 0.2); /* 부드러운 그림자로 깊이감 부여 */

            /* 버튼도 가장 마지막에 부드럽게 등장 */
            opacity: 0;
            animation: softFadeUp 1s ease-out forwards;
            animation-delay: 1.3s;
        }

        /* 버튼 호버 효과: 사용자의 행동에 부드럽게 반응 */
        .btn-primary:hover {
            background-color: #7A6A58;
            transform: translateY(-2px); /* 살짝 떠오르는 느낌 */
        }

    </style>
</head>
<body>

    <div class="container">
        <!-- 핵심 질문 -->
        <div class="question">
            지금 가장 해결하고 싶은<br>
            고민은 무엇인가요?
        </div>

        <!-- 부가 설명 (부담 낮추기) -->
        <div class="sub-text">
            아주 사소한 것이라도 괜찮아요.<br>
            당신의 이야기를 들을 준비가 되어 있습니다.
        </div>

        <!-- 행동 유도 버튼 -->
        <button class="btn-primary">
            이야기 시작하기
        </button>
    </div>

</body>
</html>
```
