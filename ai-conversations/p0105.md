# 제미나이의 응답

안녕하세요, 대표님! 1인 창업가의 길을 걷고 계신 것을 진심으로 응원합니다. 엑셀이나 시트가 낯설어도 괜찮습니다. **'기록'은 복잡한 수식이 아니라, 꾸준함이 생명**이니까요.

요청하신 **[AI 협업 프롬프트 기록 아카이브]** 양식을 설계해 드립니다. 이 시트는 나중에 대표님의 **'업무 비서 매뉴얼'**이 될 아주 중요한 자산입니다.

아래 순서대로 따라오시면 1분 만에 만드실 수 있습니다.

---

### 1. 새 구글 시트 만들기
가장 먼저, 아래 링크를 클릭하세요. 구글에 로그인된 상태라면 즉시 **새로운 빈 시트**가 생성됩니다.

👉 **[새 구글 시트 만들기 (클릭)](https://sheet.new)**

### 2. 시트 구성 항목 (헤더 작성 가이드)
새로 열린 시트의 **첫 번째 줄(1행)**에 아래 표의 **'열 이름'**을 순서대로(A1, B1, C1...) 적어주세요.

| 열 위치 | 열 이름 (Header) | 작성 가이드 (무엇을 적나요?) | 작성 예시 |
| :--- | :--- | :--- | :--- |
| **A열** | **날짜** | 작업을 진행한 날짜를 적습니다. (나중에 시계열로 복기할 때 중요합니다.) | 2024-05-21 |
| **B열** | **작업 제목** | 어떤 업무를 위해 AI를 썼는지 한눈에 알 수 있게 적습니다. | 블로그 마케팅 초안 작성 |
| **C열** | **사용 AI 모델** | 챗GPT, 클로드, 제미나이 등 어떤 도구를 썼는지 기록합니다. (모델별 성능 비교에 좋습니다.) | ChatGPT-4o |
| **D열** | **프롬프트 (명령어)** | **(핵심)** AI에게 입력한 구체적인 질문이나 명령을 복사해서 붙여넣습니다. | "30대 직장인을 타겟으로 한 생산성 도구 홍보 인스타 게시글 3개를 이모지 포함해서 써줘." |
| **E열** | **결과물 요약** | AI가 내놓은 답변 중 채택한 내용이나 핵심을 요약합니다. (전체를 다 넣기보다 쓸만한 것 위주로) | 2번째 옵션 채택 (감성적인 톤앤매너) |
| **F열** | **만족도/피드백** | 결과물이 마음에 들었는지(상/중/하), 혹은 아쉬웠던 점을 적습니다. 다음 프롬프트 수정에 도움이 됩니다. | 중 (너무 광고 티가 남, 톤 수정 필요) |
| **G열** | **비고/링크** | 해당 대화창의 공유 링크나, 참고했던 레퍼런스 자료가 있다면 적습니다. | (채팅방 공유 링크 URL) |

### 3. 초보자를 위한 시트 정리 꿀팁 (코치's Tip)

시트를 더 보기 편하게 만드는 **3가지 설정**만 딱 알려드릴게요. 이것만 해도 전문가처럼 보입니다.

1.  **첫 줄 고정하기 (스크롤 내려도 제목 보이게)**
    *   상단 메뉴에서 `보기(View)` → `고정(Freeze)` → `행 1개(1 row)`를 클릭하세요.
    *   이제 데이터가 많아져서 스크롤을 내려도 '날짜', '프롬프트' 같은 제목이 계속 보입니다.

2.  **줄 바꿈 설정 (긴 글이 잘리지 않게)**
    *   `D열(프롬프트)`과 `E열(결과물)`은 내용이 길어질 수 있습니다.
    *   D열과 E열의 알파벳 머리글을 클릭해서 선택한 뒤, 상단 도구 모음에서 **'텍스트 줄바꿈(Text wrapping)'** 아이콘(가로선이 꺾인 화살표 모양)을 찾아 **'줄바꿈(Wrap)'**을 선택하세요.

3.  **강조하기**
    *   1행(제목 줄)을 드래그해서 선택한 뒤, **'굵게(B)'** 처리하고 배경색을 연한 회색이나 노란색으로 칠해주세요. 구분이 확 됩니다.
