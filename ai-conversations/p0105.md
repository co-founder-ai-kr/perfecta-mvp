# 제미나이의 응답

반갑습니다! 구글 시트 전문가이자 당신의 업무 생산성 코치입니다.

AI와 협업한 내용을 기록으로 남기는 것은 단순히 '적는 행위'를 넘어, 나만의 **'프롬프트 자산(Asset)'**을 구축하는 훌륭한 습관입니다. 나중에 비슷한 업무를 할 때 시행착오를 획기적으로 줄여줄 것입니다.

요청하신 **[AI 프롬프트 아카이브]** 양식을 바로 만들 수 있도록 가이드를 드리겠습니다.

### 2. 시트 구성 가이드 (각 열의 역할)
새로 만든 시트의 **1행(헤더)**에 아래 항목들을 순서대로 입력하고, 각 열을 다음과 같이 활용하세요.

| 열 위치 | 항목명 (Header) | 작성 가이드 및 꿀팁 | 예시 데이터 |
| :-- | :-- | :-- | :-- |
| **A열** | **날짜** | 언제 작업했는지 기록합니다.<br>*(Tip: 셀을 선택하고 `Ctrl` + `;` 키를 누르면 오늘 날짜가 자동 입력됩니다.)* | 2024-05-20 |
| **B열** | **제목 (키워드)** | 나중에 `Ctrl` + `F`로 찾기 쉽도록 핵심 업무명을 적습니다. | [마케팅] 신제품 SNS 홍보 문구 작성 |
| **C열** | **프롬프트 (Input)** | AI에게 입력한 명령어 전체를 복사해 붙여넣습니다.<br>*(Tip: 내용이 길어질 수 있으니 서식 메뉴에서 '텍스트 줄바꿈'을 설정하세요.)* | "20대 직장인을 타겟으로 한 무선 키보드 인스타그램 홍보 문구 3가지를 이모지를 포함해 작성해 줘." |
| **D열** | **결과물 (Output)** | AI가 내놓은 답변 중 채택한 내용이나 요약본을 적습니다.<br>내용이 너무 길다면 구글 닥스 링크를 넣어도 좋습니다. | 1. 💼 퇴근 후에도 손목이 편안한... (생략) |
| **E열** | **비고 (Insight)** | 결과에 대한 만족도(상/중/하)나 수정 사항, 다음에 보완할 점을 적습니다. 이 부분이 가장 중요합니다. | 톤앤매너가 너무 딱딱함. 다음엔 '친근한 말투' 옵션 추가 필요. |

### 💡 생산성 코치의 추가 조언 (Pro Tips)

1.  **틀 고정하기:** 데이터가 쌓이면 항목명(1행)이 안 보일 수 있습니다.
    *   *방법: 상단 메뉴 [보기] > [고정] > [행 1개] 선택*
2.  **줄바꿈 설정:** C열(프롬프트)과 D열(결과물)은 내용이 깁니다. 셀 밖으로 삐져나가지 않게 설정하세요.
    *   *방법: C, D열 전체 선택 > 상단 툴바에서 [텍스트 줄바꿈(아이콘)] 선택*
3.  **배경색 지정:** 1행(헤더)의 배경색을 옅은 회색이나 파란색으로 칠하고 글자를 굵게(`Ctrl`+`B`) 하면 훨씬 가독성이 좋아집니다.

이 양식으로 꾸준히 기록하신다면, 3개월 뒤에는 당신만의 강력한 **'AI 업무 매뉴얼'**이 완성되어 있을 것입니다. 지금 바로 시작해 보세요!
