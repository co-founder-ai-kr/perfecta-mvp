# 민순의 요청

```text
* 역할: 너는 구글 시트 튜터야.
* 상황: 엑셀 함수를 전혀 모르는 문과생 창업가가 주문 내역 데이터를 가지고 있는데 메뉴 위치조차 찾지 못하고 있어.
* 과업:
  1. 가장 많이 팔린 상품 순위를 보는 법을 설명해 줘.
  2. '탐색' 기능 위치를 안내해 줘.
  3. 연인 선물과 상사 선물의 평균 가격 차이를 비교해 줘.
* 제약:
  - 어려운 함수 대신 마우스 클릭으로 해결하는 법을 알려줄 것.
  - 단계별 목록(List)으로 설명할 것.
* 출력: 단계별 가이드
* 입력: [분석하고 싶은 내용]
 (예시: 연인 선물과 상사 선물의 평균 가격 차이 비교)
```

# 제미나이의 응답

대표님, 안녕하세요! 엑셀 함수 때문에 막막하셨죠? 걱정하지 마세요.
함수(수식)를 하나도 쓰지 않고, **오직 마우스 클릭**만으로 원하시는 데이터를 뽑아내는 방법을 알려드릴게요.

구글 시트에는 **'피벗 테이블'**이라는 기능이 있는데, 이게 바로 **"데이터를 요리조리 돌려보며 요약해 주는 기계"**라고 생각하시면 됩니다.

자, 심호흡 한번 하시고 마우스만 준비해 주세요!

---

### 📍 1단계: 마법의 도구 '탐색' 버튼 위치 찾기

가장 먼저, 구글 시트가 알아서 데이터를 분석해 주는 **'탐색'** 기능이 어디 있는지부터 알려드릴게요.

1.  구글 시트 화면의 **가장 오른쪽 아래 모서리**를 봐주세요.
2.  **초록색 별 모양 아이콘(✨)**이 보이시나요?
3.  마우스를 올리면 **'탐색'**이라는 글자가 뜹니다.
    *   *팁: 이 버튼을 누르면 구글이 알아서 그래프도 그려주고 분석도 해줍니다. 급할 때 눌러보세요!*

---

### 🏆 2단계: 가장 많이 팔린 상품 순위 보기 (판매량 순위)

어떤 상품이 효자 상품인지 한눈에 확인해 봅시다.

1.  주문 내역이 있는 표 안의 아무 셀이나 클릭하세요.
2.  상단 메뉴에서 **[삽입]**을 클릭하고 **[피벗 테이블]**을 선택하세요.
3.  작은 창이 뜨면 **[만들기]** 버튼을 누르세요. (새 시트가 열립니다)
4.  이제 화면 **오른쪽**에 생긴 **'피벗 테이블 편집기'**만 보시면 됩니다.
    *   **[행]** 옆의 **[추가]** 버튼 클릭 → **'상품명'** 선택
    *   **[값]** 옆의 **[추가]** 버튼 클릭 → **'판매 수량'** (또는 주문 건수) 선택
    *   **[정렬]** 부분에서 **'오름차순'**을 **'내림차순'**으로 변경
5.  **결과:** 판매량이 가장 높은 상품이 맨 위에 오도록 순위가 매겨졌습니다!

---

### 🎁 3단계: 연인 선물 vs 상사 선물 평균 가격 비교

"누구에게 주는 선물인지"에 따라 평균 얼마를 쓰는지 비교해 볼까요?

1.  방금 만든 피벗 테이블 시트 말고, 다시 **원래 데이터가 있는 시트**로 돌아가세요.
2.  상단 메뉴 **[삽입]** → **[피벗 테이블]** → **[만들기]**를 다시 한번 눌러주세요.
3.  오른쪽 **'피벗 테이블 편집기'**에서 이렇게 클릭하세요.
    *   **[행]** 옆의 **[추가]** 버튼 클릭 → **'선물 대상'** (예: 연인, 상사 등이 적힌 항목) 선택
    *   **[값]** 옆의 **[추가]** 버튼 클릭 → **'판매 가격'** (또는 결제 금액) 선택
4.  **🚨 여기가 제일 중요해요!**
    *   방금 추가한 '판매 가격' 아래에 보면 **'SUM'** (합계)이라고 되어 있을 거예요.
    *   이걸 클릭해서 **'AVERAGE'** (평균)로 바꿔주세요.
5.  **결과:** '연인' 옆의 숫자와 '상사' 옆의 숫자를 보시면 평균 가격 차이가 한눈에 보입니다.

---

**💡 요약 꿀팁**
*   **행(Row):** "무엇을 기준으로 나눌까?" (상품별? 대상별?)
*   **값(Value):** "무엇을 계산할까?" (가격? 수량?)
*   **SUM vs AVERAGE:** 다 더할 거면 SUM, 평균을 낼 거면 AVERAGE.

이제 복잡한 함수 공부 대신, 이 **'클릭 3번'**으로 매출 분석을 시작해 보세요! 막히는 게 있으면 언제든 다시 물어봐 주세요.
