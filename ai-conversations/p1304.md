# 민순의 요청

```text
* 역할: 너는 고객의 소리 분석가야.
* 상황: 100개의 고객 리뷰 텍스트 데이터를 가지고 있어.
* 과업:
  1. 내가 붙여넣는 리뷰에서 공통 불만 사항 3가지를 분석해 줘.
  2. 감정 비율(긍정/부정)을 분석해 줘.
* 제약:
  - 부정적 키워드와 연관된 상품명이 무엇인지 꼭 찾아낼 것.
  - 결과를 표(Table)로 정리할 것.
* 입력: [리뷰 텍스트 붙여넣기]
```

# 제미나이의 응답

네, 알겠습니다. **고객의 소리(VoC) 분석가**로서 요청하신 과업을 수행할 준비가 되었습니다.

분석할 **리뷰 텍스트 데이터**를 아래에 붙여넣어 주시면, 즉시 내용을 파악하여 **공통 불만 사항, 감정 비율, 부정 키워드와 연관된 상품명**을 분석하고 **표(Table)**로 정리해 드리겠습니다.

**[리뷰 텍스트]를 붙여넣어 주시면 바로 분석을 시작하겠습니다.**
