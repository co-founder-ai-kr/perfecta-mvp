```text
* 역할: 너는 자바스크립트(JavaScript) 교육 전문 코딩 강사야.
* 상황: 내 아이디어를 직접 구현해보고 싶어.
* 과업: 
  1. 입력된 로직을 웹 사이트에서 동작하는 자바스크립트 코드로 바꿔 줘.
  2. 이 코드를 HTML 파일의 어느 위치에 넣어야 하는지 정확히 알려 줘.
* 제약:
  - 주석은 기술 용어가 아니라 비즈니스 의도로 달아 줘(예: // 여기서 고객의 지갑을 연다).
  - 복잡한 데이터베이스 연결 없이, 코드 안에 선물 목록 데이터를 직접 넣어서 처리하는 방식으로 만들어 줘.
* 입력: [비즈니스 로직 설명]
 (예시: 사용자가 예산과 관계를 입력하고 추천 버튼을 누르면, 먼저 예산 범위에 맞는 선물들을 골라낸다. 그 중에서 관계 태그와 일치하는 선물을 찾는다. 만약 적절한 선물이 없으면 ‘무난한 베스트셀러’를 대신 보여준다.)
```

# 제미나이의 응답

반갑습니다! 자바스크립트 교육 전문 강사입니다.

말씀해주신 **[선물 추천 비즈니스 로직]**을 웹 브라우저에서 바로 실행할 수 있는 코드로 변환해 드립니다. 데이터베이스 없이 코드 내부에 선물 목록을 담아서, 예산과 관계에 따라 딱 맞는 선물을 찾아주거나 베스트셀러를 제안하는 방식입니다.

아래 내용을 복사해서 바로 사용해보세요.

### 1. 완성된 코드 (HTML + JavaScript)

이 코드는 화면(HTML)과 로직(JavaScript)이 합쳐져 있습니다.

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>AI 선물 추천기</title>
    <style>
        body { font-family: sans-serif; padding: 20px; }
        .container { max-width: 400px; margin: 0 auto; }
        input, select, button { width: 100%; padding: 10px; margin-bottom: 10px; }
        #result { margin-top: 20px; font-weight: bold; color: #2c3e50; border: 1px solid #ddd; padding: 15px; }
    </style>
</head>
<body>

<div class="container">
    <h2>🎁 맞춤 선물 추천기</h2>
    
    <!-- 사용자 입력 부분 -->
    <label>예산 (원)</label>
    <input type="number" id="userBudget" placeholder="예: 50000">
    
    <label>받는 사람과의 관계</label>
    <select id="userRelation">
        <option value="friend">친구</option>
        <option value="lover">연인</option>
        <option value="parent">부모님</option>
        <option value="colleague">직장동료</option>
    </select>

    <button onclick="recommendGift()">추천 받기</button>

    <!-- 결과가 나올 공간 -->
    <div id="result">결과가 여기에 표시됩니다.</div>
</div>

<script>
    // 1. 선물 창고 (데이터베이스 대신 여기에 물건을 채워둡니다)
    const giftList = [
        { name: "스타벅스 기프티콘", price: 10000, relation: "friend", isBestseller: true },
        { name: "핸드크림 세트", price: 25000, relation: "colleague", isBestseller: false },
        { name: "고급 와인", price: 70000, relation: "friend", isBestseller: false },
        { name: "커플 잠옷", price: 60000, relation: "lover", isBestseller: false },
        { name: "명품 립스틱", price: 50000, relation: "lover", isBestseller: true },
        { name: "홍삼 세트", price: 100000, relation: "parent", isBestseller: true },
        { name: "현금 봉투", price: 200000, relation: "parent", isBestseller: true },
        { name: "무난한 디퓨저", price: 30000, relation: "colleague", isBestseller: true }
    ];

    function recommendGift() {
        // 2. 고객이 입력한 정보를 가져옵니다 (지갑 사정과 누구 줄 건지)
        const budget = parseInt(document.getElementById('userBudget').value);
        const relation = document.getElementById('userRelation').value;
        const resultBox = document.getElementById('result');

        // 예산 입력을 안 했을 경우를 대비해 잔소리(?)를 합니다.
        if (!budget) {
            resultBox.innerText = "예산을 먼저 입력해주세요!";
            return;
        }

        // 3. 1차 필터링: 고객의 지갑 사정(예산)에 맞는 물건들만 창고에서 꺼냅니다.
        const affordableGifts = giftList.filter(item => item.price <= budget);

        // 4. 2차 필터링: 예산 맞는 것 중에서 관계(태그)가 일치하는 걸 찾습니다.
        const perfectMatches = affordableGifts.filter(item => item.relation === relation);

        let finalChoice = null;

        // 5. 추천 로직 실행
        if (perfectMatches.length > 0) {
            // 조건에 딱 맞는 선물이 있다면, 그 중에서 가장 비싼(좋은) 걸 추천합니다.
            finalChoice = perfectMatches.sort((a, b) => b.price - a.price)[0];
            resultBox.innerText = `추천 결과: 딱 맞는 "${finalChoice.name}" 어떠세요? (${finalChoice.price}원)`;
        } else {
            // 6. 조건에 맞는 게 없다면, '무난한 베스트셀러'를 찾아서 제안합니다 (플랜 B 가동).
            // 단, 베스트셀러 중에서도 예산은 맞춰야 욕을 안 먹습니다.
            const bestsellers = affordableGifts.filter(item => item.isBestseller === true);
            
            if (bestsellers.length > 0) {
                finalChoice = bestsellers[0];
                resultBox.innerText = `조건에 딱 맞는 건 없지만, 베스트셀러인 "${finalChoice.name}" 추천드려요!`;
            } else {
                // 예산이 너무 적어서 베스트셀러조차 살 수 없을 때
                resultBox.innerText = "그 예산으로는 추천할 선물이 없네요... 마음을 담은 편지는 어떨까요?";
            }
        }
    }
</script>

</body>
</html>
```

### 2. 코드를 넣어야 할 정확한 위치

HTML 파일은 크게 머리(`<head>`)와 몸통(`<body>`)으로 나뉩니다. 자바스크립트는 보통 **`<body>` 태그가 끝나기 직전**에 넣는 것이 가장 좋습니다.

1.  메모장이나 코드 에디터(VS Code 등)를 엽니다.
2.  위의 코드를 통째로 복사해서 붙여넣으세요.
3.  파일 이름을 `gift.html` 처럼 뒤에 **.html**을 붙여서 저장하세요.
4.  저장된 파일을 더블 클릭하면 웹 브라우저(크롬 등)에서 바로 실행됩니다.

**핵심 포인트:**
*   `<script>` 태그 안에 있는 내용이 바로 자바스크립트 로직입니다.
*   `giftList` 변수 안에 있는 `{ name: ... }` 부분만 수정하면, 강사님 마음대로 상품 목록을 바꿀 수 있습니다.

직접 실행해보시고, 더 수정하고 싶은 부분이 있다면 언제든 말씀해주세요!
