# 민순의 요청

```text
* 역할: 너는 자바스크립트(JavaScript) 교육 전문 코딩 강사야.
* 상황: 코딩을 전혀 모르는 문과생 창업가인데, 내 아이디어를 직접 구현해보고 싶어.
* 과업: 
  1. 입력된 로직을 웹 사이트에서 동작하는 자바스크립트 코드로 바꿔 줘.
  2. 이 코드를 HTML 파일의 어느 위치에 넣어야 하는지 정확히 알려 줘.
* 제약:
  - 주석은 기술 용어가 아니라 비즈니스 의도로 달아 줘(예: // 여기서 고객의 지갑을 연다).
  - 복잡한 데이터베이스 연결 없이, 코드 안에 선물 목록 데이터를 직접 넣어서 처리하는 방식으로 만들어 줘.
* 입력: [비즈니스 로직 설명]
 (예시: 사용자가 예산과 관계를 입력하고 추천 버튼을 누르면, 먼저 예산 범위에 맞는 선물들을 골라낸다. 그 중에서 관계 태그와 일치하는 선물을 찾는다. 만약 적절한 선물이 없으면 ‘무난한 베스트셀러’를 대신 보여준다.)
```

# 제미나이의 응답

반갑습니다! 문과생 출신 창업가님, 정말 환영합니다.
아이디어를 직접 코드로 구현해 보려는 그 시도가 바로 성공한 창업가로 가는 첫걸음입니다.

복잡한 서버나 데이터베이스 없이, **HTML 파일 하나만 있으면 바로 실행되는** 가장 직관적인 코드를 짜드렸습니다. 요청하신 비즈니스 로직(예산 필터링 -> 관계 필터링 -> 없을 시 베스트셀러 추천)을 그대로 담았습니다.

---

### 1. 작성된 코드 (HTML + JavaScript)

아래 코드를 복사해서 메모장에 붙여넣고, 파일 이름을 `index.html`로 저장하세요.

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>내 아이디어 MVP: 선물 추천기</title>
    <style>
        body { font-family: sans-serif; padding: 20px; }
        .input-group { margin-bottom: 10px; }
    </style>
</head>
<body>

    <!-- [고객이 보는 화면 영역] -->
    <h1>🎁 맞춤 선물 추천 서비스</h1>
    
    <div class="input-group">
        <label>얼마까지 쓸 수 있나요? (원)</label><br>
        <input type="number" id="userBudget" placeholder="예: 30000">
    </div>

    <div class="input-group">
        <label>누구에게 줄 선물인가요?</label><br>
        <select id="userRelation">
            <option value="friend">친구</option>
            <option value="lover">연인</option>
            <option value="colleague">직장동료</option>
            <option value="family">가족</option>
        </select>
    </div>

    <!-- 고객이 이 버튼을 누르면 자바스크립트가 일을 시작합니다 -->
    <button onclick="startRecommendation()">추천받기</button>

    <hr>
    <h3>추천 결과:</h3>
    <div id="resultArea" style="font-weight: bold; color: blue;">
        결과가 여기에 나옵니다.
    </div>


    <!-- [자바스크립트 코드 영역] -->
    <!-- 여기서부터가 실제 비즈니스 로직이 돌아가는 '두뇌'입니다 -->
    <script>
        
        // 1. 우리 창고에 쌓여있는 선물 리스트 (데이터베이스 대신 여기에 직접 적어둡니다)
        const giftInventory = [
            { name: "스타벅스 기프티콘", price: 10000, tag: "colleague" },
            { name: "고급 핸드크림", price: 25000, tag: "friend" },
            { name: "무선 이어폰", price: 150000, tag: "lover" },
            { name: "홍삼 세트", price: 50000, tag: "family" },
            { name: "호텔 식사권", price: 200000, tag: "lover" },
            { name: "귀여운 잠옷", price: 30000, tag: "friend" }
        ];

        // 2. 고객이 '추천받기' 버튼을 클릭했을 때 실행되는 판매 전략
        function startRecommendation() {
            
            // 고객이 입력한 정보를 가져온다 (고객 니즈 파악)
            let budget = document.getElementById('userBudget').value;
            let relation = document.getElementById('userRelation').value;

            // [1단계 필터] 고객의 지갑 사정(예산)에 맞는 물건만 먼저 추려낸다.
            let affordableGifts = giftInventory.filter(item => item.price <= budget);

            // [2단계 필터] 예산에 맞는 것들 중에서, 관계(태그)가 딱 맞는 상품을 최종적으로 골라낸다.
            let finalMatches = affordableGifts.filter(item => item.tag === relation);

            // 결과를 보여줄 화면의 위치를 찾는다.
            let resultBox = document.getElementById('resultArea');

            // [3단계 판단] 만약 조건에 맞는 게 하나라도 있다면?
            if (finalMatches.length > 0) {
                // 고객에게 가장 먼저 발견된 딱 맞는 상품을 제안하여 구매를 유도한다.
                resultBox.innerText = "추천 상품: " + finalMatches[0].name + " (" + finalMatches[0].price + "원)";
            } 
            else {
                // [플랜 B] 만약 조건에 맞는 게 없어서 고객이 실망하고 나가려 한다면?
                // 빈손으로 보내지 말고, 가장 무난해서 실패하지 않는 '베스트셀러'를 들이민다.
                resultBox.innerText = "조건에 딱 맞는 건 없지만, 이건 어떠세요? [무난한 베스트셀러: 디퓨저 세트]";
            }
        }

    </script>
</body>
</html>
```

---

### 2. 코드를 넣어야 하는 정확한 위치

자바스크립트 코드는 HTML 파일 내에서 `<script>` 태그와 `</script>` 태그 사이에 들어갑니다.

*   **위치:** `</body>` (바디 태그가 끝나는 부분) **바로 윗줄**에 넣는 것이 가장 좋습니다.
*   **이유:** 웹 브라우저가 화면(버튼, 입력창 등)을 먼저 다 그려놓은 뒤에, 기능(자바스크립트)을 읽어들이는 것이 에러를 방지하고 속도도 빠르기 때문입니다.

위의 예시 코드에서는 제가 이미 **`<!-- [자바스크립트 코드 영역] -->`** 주석 아래에 정확한 위치를 잡아두었습니다.

### 3. 창업가님을 위한 팁

1.  **데이터 수정:** `giftInventory` 안에 있는 `{ name: "...", price: ... }` 부분만 수정하면, 창업가님이 팔고 싶은 물건으로 바로 바뀝니다.
2.  **실행 방법:** 저장한 `index.html` 파일을 더블 클릭하면 크롬이나 사파리 같은 브라우저가 열리면서 바로 테스트해볼 수 있습니다.
