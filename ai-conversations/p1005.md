# 민순의 요청

```text
* 역할: 너는 콘텐츠 마케팅 매니저야.
* 상황: 혼자서 매일 글감을 생각하느라 머리가 터질 것 같아.
* 과업: 이번 달 핵심 키워드를 주제로 4주 치 콘텐츠 발행 캘린더를 작성해 줘.
* 제약: 
  - 주말은 제외하고 평일에만 발행하는 일정으로 짜 줘.
  - [날짜, 주제, 형식, 핵심 메시지]를 포함한 표 형식으로 출력해 줘.
* 입력: [핵심 키워드]
 (예시: '감사', '선물')
```

# 제미나이의 응답

마케터님, 혼자서 매일 콘텐츠 고민하시느라 정말 고생 많으셨어요! 머리가 터질 것 같은 그 기분, 저도 너무 잘 압니다. 이제 제가 든든한 파트너가 되어 드릴게요.

입력해주신 예시 키워드인 **['감사', '선물']**을 주제로 이번 달 4주 치 콘텐츠 캘린더를 구성해 보았습니다.

이 캘린더는 **[1주차: 공감 형성] -> [2주차: 정보 제공] -> [3주차: 참여 유도] -> [4주차: 확산 및 전환]**의 흐름으로 설계되었습니다.

---

### 📅 [감사/선물] 4주 완성 콘텐츠 캘린더

| 주차 | 날짜 | 주제 | 형식 | 핵심 메시지 |
| :-- | :-- | :-- | :-- | :-- |
| **1주차** | 월 | 이번 달의 키워드: '감사'의 재정의 | 카드뉴스 | "감사는 표현할 때 비로소 완성됩니다." 감사의 중요성 환기 |
| (공감) | 화 | 선물 고를 때 가장 고민되는 순간 Best 3 | 숏폼 (릴스/쇼츠) | 누구나 겪는 선물 결정 장애 순간을 연기하여 공감대 형성 |
| | 수 | 잊지 못할 선물에 대한 에세이 | 블로그/긴 글 | 물질적인 것보다 마음이 담긴 선물이 기억에 남는 이유 |
| | 목 | 상황별 짧고 굵은 감사 인사말 모음 | 텍스트/이미지 | "뭐라고 보내지?" 고민 해결해 주는 실용적인 텍스트 템플릿 |
| | 금 | [주말 예고] 나에게 주는 선물 추천 | 이미지 슬라이드 | 한 주간 고생한 나를 위해 주말에 할 수 있는 작은 사치(휴식 등) |
| **2주차** | 월 | 가격대별 센스 있는 선물 추천 (1~3만 원대) | 카드뉴스 | 부담 없지만 센스 있다는 소리 듣는 가성비 선물 리스트 |
| (정보) | 화 | 가격대별 센스 있는 선물 추천 (5~10만 원대) | 카드뉴스 | 격식과 실용성을 모두 챙긴 중저가 선물 리스트 |
| | 수 | 선물 포장, 똥손도 금손 되는 꿀팁 | 숏폼 (릴스/쇼츠) | 다이소 재료로 고급스럽게 포장하는 초간단 방법 시연 |
| | 목 | 선물할 때 절대 하지 말아야 할 실수 3가지 | 블로그 | 받는 사람을 난처하게 만드는 선물 유형과 에티켓 정보 |
| | 금 | [큐레이션] MD가 직접 고른 '감사' 테마 아이템 | 뉴스레터/긴 글 | 진정성 있는 스토리가 담긴 브랜드나 제품 소개 |
| **3주차** | 월 | [이벤트] 고마운 사람 태그하고 커피 받기 | 이미지 (이벤트) | 댓글로 친구를 소환하여 감사의 한마디 남기기 (참여 유도) |
| (참여) | 화 | 독특한 선물 언박싱 & 리얼 반응 | 숏폼 (릴스/쇼츠) | 의외의 선물을 받았을 때의 생생한 표정과 반응 담기 |
| | 수 | 팔로워들이 추천하는 '실패 없는 선물' 모음 | 카드뉴스 (UGC) | 지난주 질문 스티커로 받은 답변들을 모아서 콘텐츠화 |
| | 목 | 감사의 마음을 전하는 손편지 쓰는 법 | 감성 이미지 | 디지털 시대에 아날로그 손편지가 주는 감동과 작성 팁 |
| | 금 | 불금에 선물하기 좋은 '기프티콘' 추천 | 텍스트/이미지 | 당장 만나지 못해도 마음을 전할 수 있는 모바일 선물 큐레이션 |
| **4주차** | 월 | 실제 고객들의 선물 후기 모음 (감동 실화) | 카드뉴스/블로그 | 우리 제품/서비스를 통해 마음을 전한 고객들의 따뜻한 사연 |
| (확산) | 화 | 이번 달, 당신은 누구에게 감사했나요? | 투표/질문 | 한 달을 돌아보며 감사했던 대상을 떠올리게 하는 질문 던지기 |
| | 수 | 급하게 선물이 필요할 때 대처법 | 숏폼 (릴스/쇼츠) | 당일 배송, 퀵 서비스 등 급박한 상황 해결 꿀팁 |
| | 목 | [월말 결산] 이달의 베스트 선물 아이템 TOP 5 | 인포그래픽 | 한 달간 반응이 가장 좋았던 아이템 데이터 기반 추천 |
| | 금 | 다음 달 키워드 스포일러 & 마무리 인사 | 이미지/텍스트 | 4주간의 여정 마무리 및 다음 달 주제에 대한 기대감 조성 |

---

**💡 마케터의 팁:**
*   **월/화**는 정보성 콘텐츠로 유입을 늘리고, **수/목**은 소통과 공감, **금**요일은 가벼운 주제나 주말을 겨냥한 콘텐츠가 반응이 좋습니다.
*   위 캘린더는 예시 키워드('감사', '선물')로 작성되었습니다. **실제로 기획하고 계신 [핵심 키워드]를 알려주시면, 그에 맞춰 다시 짜드릴 수 있습니다!**
