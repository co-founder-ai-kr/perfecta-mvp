# 민순의 요청

```text
* 역할: 너는 크롬 개발자 도구 활용 전문가야.
* 상황: 웹 페이지 화면이 하얗게 나오고 아무것도 안 떠서 멘붕이 왔어. 어디서 에러를 봐야 하는지 모르겠어.
* 과업: 
  - 개발자 도구를 여는 단축 키를 알려 줘.
  - 'Console' 탭을 찾으려면 어디를 봐야 하는지 위치를 알려 줘.
* 제약: 윈도우와 맥 단축 키를 각각 알려 줘.
* 출력: 단축 키 및 위치 설명
* 입력: [사용 중인 운영 체제]
 (예시: 윈도우)
```

# 제미나이의 응답

화면이 하얗게 나와서 많이 당황하셨겠어요! 걱정하지 마세요. **크롬 개발자 도구**를 열어서 범인을 찾는 가장 빠른 방법을 알려드릴게요.

요청하신 **단축 키**와 **Console 탭 위치**는 다음과 같습니다.

### 1. 개발자 도구 여는 단축 키

사용 중인 운영 체제에 맞춰 아래 키를 눌러보세요.

*   **Windows (윈도우):**
    *   키보드 맨 윗줄의 **`F12`** 키를 누르세요.
    *   또는 **`Ctrl` + `Shift` + `I`** 를 동시에 누르세요.
*   **Mac (맥):**
    *   **`Command(⌘)` + `Option(⌥)` + `I`** 를 동시에 누르세요.

### 2. 'Console' (콘솔) 탭 위치 찾는 법

단축 키를 누르면 화면 오른쪽이나 아래쪽에 복잡해 보이는 창(개발자 도구)이 새로 뜰 거예요. 여기서 **Console** 탭은 다음과 같이 찾을 수 있습니다.

1.  새로 뜬 창의 **가장 윗부분 메뉴 바**를 보세요.
2.  `Elements`, `Console`, `Sources`, `Network` 등의 글자가 보일 겁니다.
3.  그중 **`Console`** 이라고 적힌 탭을 클릭하세요.
    *   *만약 안 보인다면?* 메뉴 바 옆에 있는 **`>>`** 모양 아이콘을 누르면 숨겨진 메뉴 속에 `Console`이 있습니다.

**💡 전문가의 팁:**
`Console` 탭을 연 상태에서 **새로고침(F5 또는 Cmd+R)**을 한 번 해보세요. 화면이 하얗게 변하는 원인(에러)이 **빨간색 글씨**로 표시될 것입니다. 그 빨간 에러 메시지가 문제 해결의 열쇠입니다!
