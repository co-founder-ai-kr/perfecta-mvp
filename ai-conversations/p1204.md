```text
* 역할: 너는 친절한 Apps Script 튜터야.
* 상황: Apps Script 메뉴 위치도 모르는데 기초 코드를 알고 싶어.
* 과업: 구글 시트에서 데이터를 읽어서 로그(Log)에 출력하는 가장 기초적인 테스트 코드를 짜 줘.
* 제약: 코드가 작동하는 원리를 '창고 문을 연다' 같은 쉬운 비유로 주석을 달아 줘.
* 출력: Apps Script 코드
* 입력: [데이터를 읽어올 구글 시트 이름]
 (예시: '고객 목록' 시트의 A열과 B열 데이터를 읽어와 줘.)
```

# 제미나이의 응답

반가워요! Apps Script의 세계에 오신 것을 환영합니다! 🥳
메뉴 위치부터 코드 작성까지, 아주 쉽고 친절하게 알려드릴게요.

먼저, **스크립트 편집기**를 여는 방법부터 알아야겠죠?

1.  구글 시트 상단 메뉴에서 **[확장 프로그램]**을 클릭하세요.
2.  **[Apps Script]**를 선택하세요.
3.  새로운 창이 열리면, 그곳이 바로 코드를 적는 '작업실'입니다.

이제 아래 코드를 복사해서 작업실에 붙여넣고 **[저장]** (디스켓 아이콘) 후 **[실행]** 버튼을 눌러보세요.

***

### 🏗️ 기초 테스트 코드

이 코드는 **'고객 목록'** 시트의 **A열과 B열(상위 10줄)**을 읽어서 기록장에 남기는 코드입니다.

```javascript
function readCustomerData() {
  
  // 1. 거대한 창고 건물(현재 열려있는 구글 시트 파일) 안으로 들어갑니다.
  var warehouse = SpreadsheetApp.getActiveSpreadsheet();

  // 2. 창고 안에서 '고객 목록'이라고 적힌 방(시트) 문을 엽니다.
  var room = warehouse.getSheetByName('고객 목록');

  // 3. 그 방에 있는 선반 A와 B의 첫 번째 칸부터 10번째 칸까지(A1:B10)를 바라봅니다.
  // (데이터가 너무 많으면 로그가 넘치니 일단 10줄만 가져와 볼게요!)
  var shelf = room.getRange('A1:B10');

  // 4. 선반에 있는 물건들(데이터 값)을 꺼내서 상자에 담습니다.
  var items = shelf.getValues();

  // 5. 꺼낸 물건이 무엇인지 내 업무 일지(로그)에 기록합니다.
  Logger.log(items);
  
}
```

### 💡 실행 후 확인하는 법
1.  위 코드를 붙여넣고 저장한 뒤, 상단의 **▷ 실행** 버튼을 누릅니다.
2.  (처음이라면 권한 허용 팝업이 뜰 수 있어요. '권한 검토' -> 계정 선택 -> '고급' -> '이동'을 눌러 허용해주세요.)
3.  화면 아래쪽에 **[실행 로그]**라는 창이 뜨면서 시트의 데이터가 텍스트로 출력되는 것을 볼 수 있습니다!
