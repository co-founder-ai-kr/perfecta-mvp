# 민순의 요청

```text
* 역할: 너는 친절한 Apps Script 튜터야.
* 상황: Apps Script 메뉴 위치도 모르는 비개발자 1인 창업가에게 기초 코드를 알려주고 싶어.
* 과업: 구글 시트에서 데이터를 읽어서 로그(Log)에 출력하는 가장 기초적인 테스트 코드를 짜 줘.
* 제약: 코드가 작동하는 원리를 '창고 문을 연다' 같은 쉬운 비유로 주석을 달아 줘.
* 출력: Apps Script 코드
* 입력: [데이터를 읽어올 구글 시트 이름]
 (예시: '고객 목록' 시트의 A열과 B열 데이터를 읽어와 줘.)
```

# 제미나이의 응답

안녕하세요! 1인 창업가님, 만나서 반가워요. 코딩이 처음이라 막막하시죠? 걱정 마세요. 엑셀(구글 시트)을 다루실 줄 안다면 Apps Script도 금방 배우실 수 있어요.

우선 **[확장 프로그램] > [Apps Script]** 메뉴를 클릭해서 스크립트 편집기 창을 열어주세요. 그곳이 바로 우리가 작업할 공간입니다.

요청하신 대로 **'고객 목록'** 시트의 **A열(이름)**과 **B열(전화번호)** 데이터를 가져와서 확인해보는 가장 기초적인 코드를 준비했어요. 이해하기 쉽도록 **'창고에서 물건 꺼내기'**에 비유해서 설명해 드릴게요.

아래 코드를 복사해서 붙여넣고 저장(💾)해 보세요!

```javascript
function readCustomerData() {
  // 1. 현재 열려 있는 구글 시트 파일(거대한 창고 건물)에 들어갑니다.
  var app = SpreadsheetApp.getActiveSpreadsheet();

  // 2. 건물 안에서 '고객 목록'이라는 이름표가 붙은 '방(Sheet)'을 찾아서 문을 엽니다.
  var sheet = app.getSheetByName('고객 목록');

  // 3. 그 방의 선반에서 물건을 꺼낼 위치를 정합니다.
  // (2번째 줄부터 시작해서, 데이터가 있는 마지막 줄까지, A열(1번째)부터 B열(2번째)까지 잡습니다.)
  // 문법: getRange(시작행, 시작열, 몇 줄, 몇 칸)
  var lastRow = sheet.getLastRow(); // 데이터가 있는 마지막 줄 번호 확인
  var range = sheet.getRange(2, 1, lastRow - 1, 2); 

  // 4. 정해둔 선반 위에 있는 '상자(데이터)'들을 전부 꺼내옵니다.
  var data = range.getValues();

  // 5. 꺼내온 상자를 하나씩 열어서 장부(로그)에 기록합니다.
  // (상자가 없을 때까지 반복해서 확인하는 작업입니다)
  for (var i = 0; i < data.length; i++) {
    var name = data[i][0];  // 상자의 첫 번째 칸(A열) 내용물: 이름
    var phone = data[i][1]; // 상자의 두 번째 칸(B열) 내용물: 전화번호
    
    // 기록장에 적습니다. (화면 아래 '실행 로그'에서 볼 수 있어요)
    Logger.log((i + 1) + "번째 고객님: " + name + " / 연락처: " + phone);
  }
}
```

### 🏃‍♂️ 실행 방법
1.  위 코드를 붙여넣은 뒤, 상단의 **디스켓 아이콘(💾)**을 눌러 저장하세요.
2.  `readCustomerData` 함수가 선택된 상태에서 **'실행'** 버튼을 누르세요.
3.  (처음 실행하면 권한 허용 팝업이 뜰 텐데, 안심하고 허용해 주시면 됩니다.)
4.  화면 아래쪽 **'실행 로그'**라는 창에 고객 데이터가 쭉 뜨는지 확인해 보세요!

어때요? 창고에서 물건을 꺼내오는 과정과 비슷하죠? 이 코드가 익숙해지면 데이터를 수정하거나 이메일로 보내는 것도 가능해진답니다. 화이팅입니다!
