# 민순의 요청

```text
* 역할: 너는 도메인 및 DNS 설정 전문가야.
* 상황: 도메인을 샀고 네틀리파이에 연결하려고 하는데 설정이 처음이야.
* 과업: 네틀리파이에서 알려준 네임서버(Nameserver) 정보를 도메인 구입처 사이트의 어디에 입력해야 하는지 단계별로 알려 줘.
* 입력: [도메인 구입처 이름]
 (예시: 호스팅 KR, 가비아 등)
```

# 제미나이의 응답

반갑습니다! 도메인 및 DNS 설정 전문가입니다. 네틀리파이(Netlify)를 처음 사용하시는군요. 아주 훌륭한 선택입니다.

현재 **[도메인 구입처 이름]**을 구체적으로 입력해주지 않으셔서, 한국 사용자들이 가장 많이 이용하는 **'호스팅케이알(Hosting KR)'**과 **'가비아(Gabia)'** 두 곳의 설정 방법을 단계별로 정리해 드리겠습니다.

만약 다른 곳(예: 고대디, 후이즈 등)에서 구매하셨다면 구입처 이름을 말씀해 주세요. 그에 맞춰 다시 안내해 드리겠습니다.

---

### 0. 사전 준비 (네틀리파이에서 정보 확인)
먼저 네틀리파이 화면에 떠 있는 **4개의 네임서버 주소**를 복사해 두거나 창을 띄워두세요. 보통 아래와 같은 형식입니다.
*   `dns1.p0x.nsone.net`
*   `dns2.p0x.nsone.net`
*   `dns3.p0x.nsone.net`
*   `dns4.p0x.nsone.net`

---

### 1. 호스팅케이알 (Hosting KR)인 경우

호스팅케이알은 메뉴가 직관적인 편입니다.

1.  **로그인:** 호스팅케이알 사이트에 접속하여 로그인합니다.
2.  **나의 도메인:** 우측 상단 혹은 메인 메뉴에서 **[나의 도메인]**을 클릭합니다.
3.  **도메인 선택:** 구매한 도메인 이름(예: `example.com`)을 클릭하여 관리 페이지로 들어갑니다.
4.  **네임서버 관리:** 페이지 중간쯤에 있는 **[네임서버/DNS]** 탭을 클릭하거나 **[네임서버 주소 변경]** 메뉴를 찾습니다.
5.  **변경 클릭:** 현재 설정된 네임서버 목록 우측의 **[변경]** 버튼을 누릅니다.
6.  **정보 입력:**
    *   기존에 있던 호스팅케이알 네임서버 주소를 모두 지웁니다.
    *   **네틀리파이에서 받은 주소 4개**를 1차부터 4차까지 순서대로 입력합니다. (IP주소는 입력할 필요 없습니다.)
7.  **저장:** **[변경하기]** 또는 **[저장]** 버튼을 누르면 완료됩니다.

---

### 2. 가비아 (Gabia)인 경우

가비아는 '관리툴'이라는 별도 페이지로 이동해야 합니다.

1.  **로그인:** 가비아 사이트에 접속하여 로그인합니다.
2.  **My가비아:** 메인 화면 상단의 **[My가비아]**를 클릭합니다.
3.  **서비스 관리:** 이용 중인 서비스 목록에서 도메인 우측의 **[관리툴]** 버튼을 클릭합니다. (새 창이 뜰 수 있습니다.)
4.  **도메인 선택:** 관리툴 화면에서 설정하려는 도메인 이름 앞의 **체크박스(V)**를 선택합니다.
5.  **네임서버 설정:** 상단 메뉴 중 **[네임서버 설정]** 버튼을 클릭합니다.
6.  **정보 입력:**
    *   기존 가비아 네임서버가 적혀있는 칸을 지우고, **네틀리파이 네임서버 주소 4개**를 1차~4차 칸에 각각 입력합니다.
    *   (소유자 인증이 필요할 수 있습니다. 휴대전화나 이메일 인증을 진행하세요.)
7.  **적용:** 하단의 **[적용]** 또는 **[저장]** 버튼을 누르면 완료됩니다.

---

### 💡 전문가의 팁 (주의사항)

1.  **반영 시간 (전파 시간):** 네임서버를 변경했다고 해서 바로 접속이 되지 않을 수 있습니다. 전 세계의 네트워크에 정보가 퍼지는 데 **짧게는 10분, 길게는 24시간**까지 걸릴 수 있습니다. (보통 1시간 이내에 됩니다.)
2.  **기존 정보 삭제:** 반드시 기존에 적혀 있던 구입처의 네임서버 주소는 **모두 삭제**하고 네틀리파이 주소만 남겨야 합니다. 섞어서 쓰면 오류가 발생합니다.
3.  **HTTPS 설정:** 네임서버가 정상적으로 연결되면, 네틀리파이에서 자동으로 보안 인증서(HTTPS)를 발급해 줍니다. 연결 직후에는 '주의 요함'이 뜰 수 있으나 조금 기다리면 자물쇠 아이콘으로 바뀝니다.

**혹시 다른 구입처라면 이름을 알려주세요! 바로 위치를 찾아드리겠습니다.**
