# 민순의 요청

```text
* 역할: 너는 데이터 구조화 전문 백엔드 개발자야.
* 상황: 엑셀에 정리된 상품 정보를 자바스크립트 코드에 직접 넣어서 쓰고 싶어.
* 과업: 아래 텍스트 데이터를 자바스크립트에서 쓸 수 있는 JSON 배열 형식으로 변환해 줘.
* 제약: 
  - 변수명은 'products'로 해 줘.
  - 가격은 숫자만 남기고, 상품명은 문자열로 처리해 줘.
* 출력: JSON 데이터 코드
* 입력: [엑셀에서 복사한 데이터 붙여넣기]
```

# 제미나이의 응답

데이터가 입력되지 않았습니다. 엑셀에서 복사한 내용을 답장으로 붙여넣어 주시면, 요청하신 **제약 사항(변수명 `products`, 가격은 숫자형 변환)** 에 맞춰 즉시 코드를 작성해 드리겠습니다.

**예시)**
만약 아래와 같은 데이터를 주시면:
```text
사과    1,000원
배      2,500원
포도    $5.00
```

**결과)**
아래와 같이 변환해 드립니다:
```javascript
const products = [
  { name: "사과", price: 1000 },
  { name: "배", price: 2500 },
  { name: "포도", price: 5 }
];
```

**변환할 데이터를 붙여넣어 주세요.**
