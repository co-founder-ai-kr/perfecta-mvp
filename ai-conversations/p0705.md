# 민순의 요청

```text
* 역할: 너는 데이터 구조화 전문 백엔드 개발자야.
* 상황: 엑셀에 정리된 상품 정보를 자바스크립트 코드에 직접 넣어서 쓰고 싶어.
* 과업: 아래 텍스트 데이터를 자바스크립트에서 쓸 수 있는 JSON 배열 형식으로 변환해 줘.
* 제약:
  - 변수명은 'products'로 해 줘.
  - 가격 데이터에 있는 쉼표(,)와 원(₩) 단위는 모두 제거하고, 계산 가능한 순수 숫자(Integer) 형식으로 변환해 줘.
  - 상품명은 문자열로 처리해 줘.
* 출력: JSON 데이터 코드
* 입력: [엑셀에서 복사한 데이터 붙여넣기]
```

# 제미나이의 응답

입력란에 **엑셀 데이터**가 포함되지 않았습니다.

변환할 상품 정보(상품명, 가격 등)를 **답장으로 붙여넣어 주시면**, 요청하신 제약 조건에 맞춰 즉시 자바스크립트 코드로 변환해 드리겠습니다.

데이터를 붙여넣어 주시면 아래와 같은 로직으로 처리됩니다:

1.  **변수명:** `products`
2.  **가격 처리:** `₩`, `,`, `원` 제거 후 `Number` 타입으로 변환
3.  **결과물:** JSON 배열

**[예시]**
만약 아래와 같이 데이터를 주시면:
```text
기계식 키보드    159,000원
무선 마우스      ₩89,000
```

다음과 같이 변환해 드립니다:
```javascript
const products = [
  {
    "name": "기계식 키보드",
    "price": 159000
  },
  {
    "name": "무선 마우스",
    "price": 89000
  }
];
```

**변환할 데이터를 붙여넣어 주세요.**
