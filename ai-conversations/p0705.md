# 민순의 요청

```text
* 역할: 너는 데이터 구조화 전문 백엔드 개발자야.
* 상황: 서버나 DB를 다룰 줄 몰라서 엑셀에 정리된 상품 정보를 자바스크립트 코드에 직접 넣어서 쓰고 싶어.
* 과업: 아래 텍스트 데이터를 자바스크립트에서 쓸 수 있는 JSON 배열 형식으로 변환해 줘.
* 제약: 
  - 변수명은 'products'로 해 줘.
  - 가격은 숫자만 남기고, 상품명은 문자열로 처리해 줘.
* 출력: JSON 데이터 코드
* 입력: [엑셀에서 복사한 데이터 붙여넣기]
```

# 제미나이의 응답

네, 알겠습니다. 데이터 구조화 전문 백엔드 개발자입니다.

현재 입력란에 **실제 엑셀 데이터가 포함되지 않았습니다.**

변환하고 싶은 데이터를 **답글로 붙여넣어 주시면**, 요청하신 대로 `products` 변수명에 가격은 숫자로, 상품명은 문자열로 깔끔하게 정리된 JSON 코드를 바로 작성해 드리겠습니다.

**(예시: 아래와 같이 데이터를 주시면 됩니다)**
```text
사과    1,000원
배      2,500
포도    3,000원
```

데이터를 붙여넣어 주시면 바로 작업 시작하겠습니다.
