# 민순의 요청

```text
* 역할: 너는 HTML 구조 설계 전문 개발자야.
* 상황: 디자인은 나중에 하고 일단 화면에 어떤 내용이 들어갈지 뼈대부터 잡고 싶어.
* 과업: 내가 묘사하는 대로 HTML 구조를 잡아 줘.
* 제약: 
  - 디자인(CSS)보다 HTML 구조 잡는 데 집중해 줘. 
  - 나중에 내가 코드를 봤을 때 어디가 제목이고 어디가 버튼인지 쉽게 알아볼 수 있게 짜 줘.
* 출력: HTML 코드
* 입력: [화면 구성 묘사]
 (예시: 맨 위에 제목 넣고, 그 아래 예산 슬라이더랑 상황 선택 버튼들을 배치해 줘.)
```

# 제미나이의 응답

네, 알겠습니다. **HTML 구조 설계 전문 개발자**입니다.

디자인 요소를 배제하고, **시맨틱(Semantic) 태그**와 **주석**을 활용하여 나중에 코드를 봤을 때 어떤 영역인지 직관적으로 알 수 있도록 뼈대를 잡아드리겠습니다.

현재 입력란에 구체적인 묘사가 없으므로, **제시해주신 예시(제목, 예산 슬라이더, 상황 선택 버튼)**를 바탕으로 먼저 코드를 작성해 보여드리겠습니다.

이 형식이 마음에 드신다면, 원하시는 **[화면 구성 묘사]**를 입력해 주세요. 바로 작성해 드리겠습니다.

***

### [예시 입력에 대한 HTML 구조]

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>화면 구조 초안</title>
</head>
<body>

    <!-- ================================
         1. 헤더 영역 (제목)
    ================================ -->
    <header>
        <!-- 메인 타이틀 -->
        <h1>서비스 또는 페이지 제목</h1>
        <!-- 부가 설명 (필요시) -->
        <p>이 페이지의 목적에 대한 간단한 설명</p>
    </header>

    <hr> <!-- 구분을 위한 가로선 (구조 확인용) -->

    <!-- ================================
         2. 메인 콘텐츠 영역
    ================================ -->
    <main>

        <!-- 섹션 A: 예산 설정 -->
        <section id="budget-section">
            <h2>예산 설정</h2>
            
            <!-- 슬라이더 컨트롤 -->
            <div>
                <label for="budget-range">예산 범위:</label>
                <input type="range" id="budget-range" name="budget" min="0" max="100000" step="1000">
                <!-- 사용자가 선택한 값을 보여주는 영역 -->
                <span>선택 금액: <output>0</output>원</span>
            </div>
        </section>

        <br> <!-- 간격 (구조 확인용) -->

        <!-- 섹션 B: 상황 선택 -->
        <section id="situation-section">
            <h2>상황 선택</h2>
            <p>어떤 상황인지 선택해 주세요.</p>

            <!-- 버튼 그룹 -->
            <div class="button-group">
                <button type="button">데이트</button>
                <button type="button">친구 모임</button>
                <button type="button">가족 식사</button>
                <button type="button">비즈니스 미팅</button>
            </div>
        </section>

        <br> <!-- 간격 (구조 확인용) -->

        <!-- 섹션 C: 제출/완료 액션 -->
        <section id="action-section">
            <button type="submit">설정 완료 및 검색</button>
        </section>

    </main>

    <!-- ================================
         3. 푸터 영역 (하단 정보)
    ================================ -->
    <footer>
        <small>&copy; 2024 Company Name. All rights reserved.</small>
    </footer>

</body>
</html>
```

**이제 원하시는 화면 구성을 묘사해 주시면, 위와 같이 명확한 구조로 코드를 짜드리겠습니다.**
