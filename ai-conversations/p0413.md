# 민순의 요청

```text
* 역할: 너는 구글 애널리틱스 전문가야.
* 상황: 웹 사이트 방문자 수를 추적하고 싶은데 애널리틱스 사용이 처음이야.
* 과업: 
  1. 구글 애널리틱스 측정 ID 발급받는 방법을 알려 줘.
  2. 이 ID를 연동하기 위한 추적 코드를 어디서 복사하는지 알려 줘. 
  3. 이 추적 코드를 index.html 파일의 어느 위치에 붙여넣어야 하는지 정확히 알려 줘.
* 제약: 구글 애널리틱스 접속 주소부터 시작해서 단계별로 알려 줘.
* 출력: 단계별 실행 가이드
```

# 제미나이의 응답

반갑습니다. 구글 애널리틱스(GA4) 전문가입니다.

처음 시작하시는 분들도 쉽게 따라 하실 수 있도록, 접속부터 코드 삽입까지 **단계별 실행 가이드**를 작성해 드립니다. 현재 표준인 **GA4(Google Analytics 4)** 기준으로 설명합니다.

---

### 1단계: 구글 애널리틱스 계정 생성 및 측정 ID 발급

가장 먼저 구글 애널리틱스에 가입하고 웹사이트 정보를 등록하여 고유 ID를 받아야 합니다.

1.  **구글 애널리틱스 접속**:
    *   웹 브라우저 주소창에 `https://analytics.google.com/`을 입력하고 접속합니다.
    *   구글 계정으로 로그인합니다.
2.  **계정 생성 시작**:
    *   처음이라면 **[측정 시작]** 버튼을 클릭합니다.
    *   (이미 계정이 있다면 좌측 하단 **톱니바퀴 아이콘(관리)** > **[+ 만들기]** > **[계정]**을 선택합니다.)
3.  **계정 설정**:
    *   **계정 이름**: 회사명이나 프로젝트 이름을 입력합니다.
    *   데이터 공유 설정은 기본값대로 두고 **[다음]**을 클릭합니다.
4.  **속성 설정**:
    *   **속성 이름**: 웹사이트 이름(예: 내 포트폴리오)을 입력합니다.
    *   **보고 시간대**: '대한민국'으로 변경합니다.
    *   **통화**: '대한민국 원(KRW)'으로 변경합니다.
    *   **[다음]**을 클릭합니다.
5.  **비즈니스 세부정보**:
    *   업종 카테고리와 비즈니스 규모를 선택하고 **[다음]**을 클릭합니다.
    *   비즈니스 목표(예: 리드 생성, 브랜드 인지도 등)를 선택하고 **[만들기]**를 클릭합니다.
    *   약관 동의 창이 뜨면 '대한민국' 선택 후 체크박스에 동의하고 **[동의함]**을 누릅니다.
6.  **데이터 수집 플랫폼 선택**:
    *   데이터 수집 시작 화면에서 **[웹]**을 선택합니다.
7.  **데이터 스트림 설정 (중요)**:
    *   **웹사이트 URL**: `https://`를 제외한 주소(예: `www.mysite.com`)를 입력합니다. (아직 도메인이 없다면 로컬 테스트용 주소나 임시 주소를 넣어도 됩니다.)
    *   **스트림 이름**: 사이트 식별 이름(예: 메인 웹사이트)을 입력합니다.
    *   **[스트림 만들기]** 버튼을 클릭합니다.
8.  **측정 ID 확인**:
    *   스트림이 생성되면 화면 상단에 **`G-`로 시작하는 코드**가 보입니다. 이것이 **측정 ID**입니다. (예: `G-A1B2C3D4E5`)

---

### 2단계: 추적 코드(Google 태그) 복사하기

측정 ID만으로는 추적이 불가능하며, 자바스크립트 코드를 복사해야 합니다.

1.  스트림을 만들면 자동으로 **'웹 스트림 세부정보'** 화면이 뜹니다. (만약 창을 닫았다면: **관리(톱니바퀴) > 데이터 수집 및 수정 > 데이터 스트림 > 해당 스트림 클릭**)
2.  화면 하단 혹은 중간에 **[태그 안내 보기]** 버튼을 클릭합니다.
3.  새 창이 뜨면 상단 탭에서 **[직접 설치] (Install manually)** 탭을 클릭합니다.
4.  `<!-- Google tag (gtag.js) -->`로 시작하는 코드 블록이 나타납니다.
5.  코드 상단의 **복사 아이콘**을 클릭하여 코드를 클립보드에 복사합니다.

---

### 3단계: index.html 파일에 코드 붙여넣기

이제 복사한 코드를 실제 웹사이트 파일에 심어야 합니다.

1.  웹사이트의 소스 코드가 있는 폴더에서 **`index.html`** 파일을 엽니다. (VS Code, 메모장 등 편집기 사용)
2.  HTML 코드 내에서 **`<head>`** 태그를 찾습니다.
3.  **`<head>` 태그 바로 다음 줄**, 혹은 **`</head>`(닫는 태그) 바로 윗줄**에 복사한 코드를 붙여넣습니다.
    *   *전문가 팁: 데이터를 가장 먼저 수집하기 위해 `<head>` 태그 중에서도 가능한 위쪽에 붙여넣는 것을 권장합니다.*

**[예시 코드]**

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>내 웹사이트</title>

    <!-- 여기에 붙여넣으세요: Google tag (gtag.js) -->
    <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
    <script>
      window.dataLayer = window.dataLayer || [];
      function gtag(){dataLayer.push(arguments);}
      gtag('js', new Date());

      gtag('config', 'G-XXXXXXXXXX');
    </script>
    <!-- 구글 애널리틱스 코드 끝 -->

    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>환영합니다!</h1>
    <!-- 본문 내용 -->
</body>
</html>
```

4.  파일을 **저장**하고, 웹 서버에 **배포(업로드)**합니다.

---

### 마무리: 잘 설치되었는지 확인하기

1.  웹사이트에 접속합니다.
2.  구글 애널리틱스 화면으로 돌아와서 좌측 메뉴의 **[보고서] > [실시간]**을 클릭합니다.
3.  현재 방문자 수가 **1** 이상으로 잡힌다면 성공적으로 연동된 것입니다. (데이터가 뜨는 데 약 10~30초 정도 걸릴 수 있습니다.)

이제 모든 준비가 끝났습니다! 데이터 분석의 세계에 오신 것을 환영합니다.
