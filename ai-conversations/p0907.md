# 민순의 요청

```text
* 역할: 너는 구글 시트 시각화 전문가야.
* 상황: 구글 시트에 쌓인 데이터로 지난 주 대비 이번 주 변화를 보고 싶어.
* 과업: 엑셀 함수를 몰라도 되게 메뉴 클릭 순서대로 꺾은선 그래프 만드는 법을 알려 줘.
* 제약: '삽입' 메뉴의 어디를 눌러야 하는지, 차트 편집기에서 X축과 Y축을 어떻게 설정하는지 마우스 클릭 단위로 설명해 줘.
* 출력: 차트 생성 가이드
* 입력: [데이터 구조 설명]
 (예시: A열 날짜, B열 신청자 수)
```

# 제미나이의 응답

반갑습니다. 구글 시트 시각화 전문가입니다.

함수를 전혀 사용하지 않고, 오직 **마우스 클릭**만으로 지난주와 이번 주의 데이터 변화를 한눈에 파악할 수 있는 **꺾은선 그래프(Line Chart)** 생성 가이드를 작성해 드립니다.

가장 일반적인 데이터 구조인 **[A열: 날짜 / B열: 수치(예: 신청자 수)]**를 기준으로 설명합니다.

---

### 📈 차트 생성 가이드: 지난주 vs 이번 주 변화 한눈에 보기

#### 1단계: 데이터 선택하기
가장 먼저 차트로 만들 데이터를 마우스로 잡아야 합니다.

1.  데이터가 있는 시트를 엽니다.
2.  **A열(날짜)**과 **B열(수치)**의 데이터가 있는 영역을 마우스로 드래그하여 **블록 지정(선택)**합니다.
    *   *팁: 제목(헤더)인 1행부터 데이터 끝까지 포함해서 선택하세요.*

#### 2단계: 차트 메뉴 진입하기 ('삽입' 메뉴)
이제 빈 캔버스에 차트를 불러옵니다.

1.  구글 시트 상단 메뉴바에서 **[삽입]**을 클릭합니다.
2.  나오는 메뉴 리스트 중간쯤에 있는 **[차트]**를 클릭합니다.
    *   *결과: 화면 중앙에 그래프가 생기고, 화면 오른쪽에는 '차트 편집기'라는 창이 열립니다.*

#### 3단계: 차트 편집기 설정 (X축, Y축 잡기)
오른쪽에 열린 **'차트 편집기'** 창에서 **[설정]** 탭을 조작합니다. 구글 시트가 자동으로 추천해주지만, 정확하게 확인해야 합니다.

1.  **차트 유형**
    *   '차트 유형' 아래의 드롭다운 메뉴를 클릭합니다.
    *   '선' 카테고리에 있는 첫 번째 아이콘 **[꺾은선형 차트]**를 클릭합니다. (지그재그 선 모양)
2.  **X축 (가로축: 날짜)**
    *   'X축' 항목을 봅니다. 여기에 **[날짜]** (A열 제목)가 들어가 있는지 확인합니다.
    *   *만약 비어있다면:* 'X축' 글자 오른쪽의 네모난 아이콘(데이터 범위 선택)을 클릭 → 시트에서 A열 데이터 드래그 → [확인] 클릭.
3.  **계열 (세로축: 수치)**
    *   '계열' 항목을 봅니다. 여기에 **[신청자 수]** (B열 제목)가 들어가 있는지 확인합니다.
    *   *만약 비어있다면:* '계열' 글자 옆의 **[계열 추가]** 클릭 → 시트에서 B열 데이터 드래그 → [확인] 클릭.

#### 4단계: 시각화 전문가의 '한 끗' (보기 좋게 다듬기)
그래프는 그려졌지만, 변화를 더 확실히 보기 위해 점과 숫자를 표시합니다. **'차트 편집기'** 창에서 **[맞춤설정]** 탭을 클릭하세요.

1.  **데이터 포인트(점) 찍기**
    *   **[계열]** 메뉴를 클릭하여 펼칩니다.
    *   아래로 조금 내려가서 **'포인트 크기'** 항목을 찾습니다.
    *   '없음'으로 되어 있는 것을 **[7px]** 또는 **[10px]**로 변경합니다.
    *   *효과: 꺾이는 부분마다 점이 생겨서 날짜별 위치를 정확히 알 수 있습니다.*
2.  **숫자 표시하기 (데이터 라벨)**
    *   방금 설정한 곳 바로 아래에 있는 **[데이터 라벨]** 체크박스를 **클릭(체크)**합니다.
    *   *효과: 그래프 선 위에 정확한 숫자가 표시되어, 지난주 대비 몇 명이 늘고 줄었는지 바로 보입니다.*

---

### ✅ 완료
이제 차트가 완성되었습니다.
*   그래프의 **기울기**를 통해 지난주보다 이번 주에 상승세인지 하락세인지 직관적으로 파악할 수 있습니다.
*   차트의 빈 공간을 클릭하고 드래그하여 원하는 위치로 옮기시면 됩니다.
