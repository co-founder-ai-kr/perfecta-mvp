# 민순의 요청

```text
* 역할: 너는 구글 시트 시각화 전문가야.
* 상황: 구글 시트에 쌓인 데이터(A열: 날짜, B열: 신청자 수)로 지난 주 대비 이번 주 변화를 보고 싶어.
* 과업: 엑셀 함수를 몰라도 되게 메뉴 클릭 순서대로 꺾은선 그래프 만드는 법을 알려 줘.
* 제약: '삽입' 메뉴의 어디를 눌러야 하는지, 차트 편집기에서 X축과 Y축을 어떻게 설정하는지 마우스 클릭 단위로 설명해 줘.
* 출력: 차트 생성 가이드
* 입력: [데이터 구조 설명]
 (예시: A열 날짜, B열 신청자 수)
```

# 제미나이의 응답

반갑습니다! 구글 시트 시각화 전문가입니다.

함수를 전혀 사용하지 않고, 오직 **마우스 클릭**만으로 지난 주와 이번 주의 신청자 수 변화를 한눈에 파악할 수 있는 **꺾은선 그래프** 생성 가이드를 알려드리겠습니다.

그대로 따라 하시면 됩니다.

---

### 📈 [차트 생성 가이드] 지난 주 vs 이번 주 추세 비교하기

#### 1단계: 데이터 선택하기
가장 먼저 차트로 만들 데이터를 선택해야 합니다.

1.  마우스로 **A열(날짜)**과 **B열(신청자 수)**의 데이터가 있는 영역을 드래그하여 블록으로 지정합니다.
    *   *(팁: 제목인 '날짜', '신청자 수' 텍스트까지 포함해서 선택하세요.)*

#### 2단계: 차트 메뉴 불러오기
이제 선택한 데이터를 바탕으로 차트를 생성합니다.

1.  구글 시트 상단 메뉴바에서 **[삽입]**을 클릭합니다.
2.  나오는 메뉴 리스트 중 중간쯤에 있는 **[차트]**를 클릭합니다.
    *   *화면에 자동으로 차트가 생성되고, 오른쪽에 '차트 편집기'라는 창이 열립니다.*

#### 3단계: 차트 편집기 설정 (핵심)
오른쪽에 열린 **'차트 편집기'** 창에서 설정을 확인하고 변경합니다. (만약 창이 닫혔다면 차트를 더블 클릭하세요.)

**[설정] 탭에서 할 일:**

1.  **차트 유형**: 드롭다운 메뉴를 클릭하여 **[꺾은선 차트]**를 선택합니다. (선 모양 아이콘)
2.  **X축**: 여기가 '날짜'로 되어 있는지 확인합니다.
    *   만약 비어 있다면 클릭 후 **[날짜]**를 선택합니다.
3.  **계열**: 여기가 '신청자 수'로 되어 있는지 확인합니다.
    *   만약 비어 있다면 클릭 후 **[신청자 수]**를 선택합니다.
4.  **맨 아래 체크박스**:
    *   **[2행을 헤더로 사용]** (또는 1행)에 체크가 되어 있는지 확인합니다. (제목줄을 데이터로 인식하지 않게 하기 위함입니다.)

#### 4단계: 그래프 보기 좋게 다듬기 (선택 사항)
변화 추이를 더 확실하게 보기 위해 점을 찍어보겠습니다.

1.  차트 편집기 상단의 **[맞춤설정]** 탭을 클릭합니다.
2.  **[계열]** 메뉴를 클릭하여 펼칩니다.
3.  아래쪽으로 조금 스크롤을 내려 **[점 크기]** 항목을 찾습니다.
4.  '없음'으로 되어 있는 것을 **[7px]** 정도로 변경합니다.
    *   *이제 꺾은선 위에 점이 생겨서 날짜별 위치를 정확히 알 수 있습니다.*
5.  바로 아래 **[데이터 라벨]** 체크박스를 클릭합니다.
    *   *그래프 위에 정확한 숫자가 표시되어 몇 명인지 바로 보입니다.*
