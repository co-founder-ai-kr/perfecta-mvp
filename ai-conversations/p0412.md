* 상황: 랜딩 페이지에 고객 정보를 받는 구글 폼을 넣고 싶은데 HTML을 몰라.
* 과업: 구글 폼의 'iframe' 코드를 웹페이지 하단에 삽입해 줘. 디자인이 깨지지 않게 가운데 정렬해 줘.

# 제미나이의 응답

네, 알겠습니다! 웹 개발자로서 깔끔하고 모바일에서도 잘 보이는 코드를 작성해 드릴게요.

아직 구글 폼의 `iframe` 코드를 주지 않으셨지만 괜찮습니다. **아래 코드의 주석 처리된 부분에 복사하신 코드를 붙여넣기만 하면 바로 중앙 정렬되도록** 틀을 짜드렸습니다.

이 코드를 웹페이지의 하단(예: `<footer>` 태그 위나 `</body>` 태그 바로 앞)에 복사해서 넣으세요.

### 📋 HTML/CSS 코드

```html
<!-- 구글 폼 섹션 시작 -->
<style>
  /* 구글 폼을 감싸는 컨테이너 스타일 */
  .google-form-wrapper {
    display: flex;
    justify-content: center; /* 가로 중앙 정렬 */
    align-items: center;     /* 세로 중앙 정렬 */
    width: 100%;             /* 전체 너비 사용 */
    padding: 40px 20px;      /* 상하 40px, 좌우 20px 여백 */
    box-sizing: border-box;  /* 패딩 포함 너비 계산 */
    background-color: #f9f9f9; /* (선택사항) 배경색을 연한 회색으로 지정 */
  }

  /* 구글 폼 iframe 반응형 처리 */
  .google-form-wrapper iframe {
    max-width: 100%; /* 모바일에서 폼이 화면을 뚫고 나가지 않게 함 */
    border: none;    /* 테두리 제거 */
    box-shadow: 0 4px 10px rgba(0,0,0,0.1); /* (선택사항) 살짝 그림자 효과 */
    border-radius: 8px; /* (선택사항) 모서리 둥글게 */
  }
</style>

<div class="google-form-wrapper">
  
  <!-- ▼▼▼ 아래에 구글 폼 iframe 코드를 붙여넣으세요 ▼▼▼ -->
  
  <iframe src="https://docs.google.com/forms/..." width="640" height="800" frameborder="0" marginheight="0" marginwidth="0">로드 중…</iframe>
  
  <!-- ▲▲▲ 여기에 구글 폼 iframe 코드를 붙여넣으세요 ▲▲▲ -->

</div>
<!-- 구글 폼 섹션 끝 -->
```

---

### 💡 사용 방법 및 팁

1.  **코드 붙여넣기:** 위 코드에서 `<!-- ▼▼▼ ... -->` 와 `<!-- ▲▲▲ ... -->` 사이에 있는 예시 `iframe` 태그를 지우고, 고객님이 구글 폼에서 복사한 코드를 그대로 붙여넣으세요.
2.  **디자인 설명:**
    *   **중앙 정렬:** `display: flex`와 `justify-content: center`를 사용하여 화면 크기에 상관없이 항상 가운데에 오도록 했습니다.
    *   **모바일 대응:** `max-width: 100%`를 추가하여 스마트폰 같은 작은 화면에서도 폼이 잘리지 않고 화면 너비에 맞춰지도록 했습니다.
    *   **여백:** 위아래로 답답하지 않게 여백(`padding`)을 주었습니다.

혹시 구글 폼 코드를 댓글로 주시면 제가 직접 합쳐서 완성된 코드를 다시 드릴 수도 있습니다! 편하게 말씀해 주세요.
