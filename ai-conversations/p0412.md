# 제미나이의 응답

반갑습니다! HTML/CSS 전문 웹 개발자입니다.

랜딩 페이지 하단에 구글 폼을 깔끔하게 넣고 싶으시군요. 보내주실 구글 폼 `iframe` 코드를 감싸서 **화면 정중앙에 배치**하고, **모바일에서도 잘 보이도록(반응형)** 만드는 코드를 작성해 드립니다.

아래 코드를 복사해서 웹페이지의 원하시는 위치(보통 `</body>` 태그 바로 위쪽)에 붙여넣기만 하시면 됩니다.

### 1. 완성된 HTML/CSS 코드

```html
<!-- 구글 폼 섹션 시작 -->
<style>
  /* 구글 폼을 감싸는 컨테이너 스타일 */
  .google-form-container {
    display: flex;
    justify-content: center; /* 가로 중앙 정렬 */
    align-items: center;     /* 세로 중앙 정렬 */
    width: 100%;             /* 전체 너비 사용 */
    padding: 40px 20px;      /* 위아래, 양옆 여백 */
    box-sizing: border-box;  /* 여백 포함 너비 계산 */
    background-color: #f9f9f9; /* (선택사항) 배경색 - 필요 없으면 지우세요 */
  }

  /* 구글 폼 iframe 반응형 스타일 */
  .google-form-container iframe {
    max-width: 100%; /* 모바일에서 화면 밖으로 튀어나가지 않게 함 */
    border: none;    /* 테두리 제거 */
    box-shadow: 0 4px 10px rgba(0,0,0,0.1); /* (선택사항) 살짝 그림자 효과 */
    border-radius: 8px; /* (선택사항) 모서리 둥글게 */
  }
</style>

<div class="google-form-container">
  
  <!-- ▼▼▼ 아래에 구글 폼에서 복사한 iframe 코드를 붙여넣으세요 ▼▼▼ -->
  
  <iframe src="https://docs.google.com/forms/..." width="640" height="800" frameborder="0" marginheight="0" marginwidth="0">로드 중…</iframe>
  
  <!-- ▲▲▲ 여기까지 ▲▲▲ -->

</div>
<!-- 구글 폼 섹션 끝 -->
```

---

### 2. 적용 방법

1.  위의 코드 박스 내용을 복사합니다.
2.  **`<!-- ▼▼▼ 아래에 ... -->`** 라고 적힌 부분과 **`<!-- ▲▲▲ 여기까지 ... -->`** 사이에 있는 예시 `iframe` 코드를 지웁니다.
3.  그 자리에 고객님이 구글 폼에서 복사해 온 **실제 `iframe` 코드**를 붙여넣습니다.
4.  완성된 전체 코드를 랜딩 페이지 HTML 파일의 하단(보통 `<footer>` 위나 `</body>` 바로 앞)에 삽입합니다.

### 3. 개발자의 팁 (추가 설명)
*   **반응형 처리:** 구글 폼은 기본적으로 고정된 너비(`width="640"`)를 가지고 있습니다. 제가 추가한 CSS(`max-width: 100%`) 덕분에 스마트폰처럼 좁은 화면에서도 폼이 잘리지 않고 화면 크기에 맞춰 자동으로 줄어듭니다.
*   **디자인:** 배경색이나 그림자 효과를 넣어두었습니다. 만약 랜딩 페이지 디자인과 어울리지 않는다면 `<style>` 태그 안의 `background-color`나 `box-shadow` 부분을 삭제하시면 됩니다.
