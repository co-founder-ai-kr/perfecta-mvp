# 민순의 요청

```text
* 역할: 너는 제미나이 캔버스 코드를 검수하는 꼼꼼한 코드 리뷰어야.
* 상황: 비개발자 1인 창업가가 작성된 코드가 내 비즈니스 의도대로 구현되었는지 확인하고 싶어 해.
* 과업: 이 코드를 한 줄 한 줄 분석해서, 비개발자인 내가 이해할 수 있게 비즈니스적으로 어떤 역할인지 한글 주석을 달아서 다시 보여 줘.
* 제약: 코드의 각 줄에 한글 주석을 달아 줘.
* 출력: 주석이 달린 코드 블록
* 입력: [검증할 코드 붙여넣기]
 (예시: (여기에 HTML 코드 전체를 붙여넣으세요. 예: `<button onclick="alert('Hello');">클릭</button>`))
```

# 제미나이의 응답

네, 알겠습니다! 비개발자 1인 창업가님의 비즈니스 의도가 코드에 잘 반영되었는지 꼼꼼하게 확인해 드릴 준비가 되었습니다.

**검증이 필요한 코드를 아래에 붙여넣어 주시면**, 즉시 한 줄 한 줄 분석하여 비즈니스 관점의 주석을 달아드리겠습니다.

(예시로 들어주신 코드를 바탕으로, 제가 어떻게 결과물을 드릴지 먼저 보여드리겠습니다.)

***

**[예시 출력]**

```html
<!-- [비즈니스 요소: 액션 버튼] -->
<!-- 고객이 화면에서 상호작용할 수 있는 버튼을 생성하는 시작점입니다. -->
<button 
    /* [트리거 설정] 고객이 이 버튼을 '클릭'했을 때 시스템이 반응하도록 설정합니다. */
    onclick="
        /* [시스템 반응] 화면 중앙에 알림 팝업창을 띄웁니다. */
        /* 고객에게 'Hello'라는 환영 메시지를 전달하는 기능입니다. */
        alert('Hello');
    ">
    
    <!-- [버튼 텍스트] 고객의 눈에 실제로 보이는 버튼의 이름(라벨)입니다. -->
    클릭
</button>
```
