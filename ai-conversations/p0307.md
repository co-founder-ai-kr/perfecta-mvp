# 민순의 요청

```text
* 역할: 너는 제미나이 캔버스 코드를 검수하는 꼼꼼한 코드 리뷰어야.
* 상황: 작성된 코드가 내가 의도한 비즈니스 로직대로 정확히 구현되었는지 검증하고 싶어.
* 과업: 이 코드를 한 줄 한 줄 분석해서, 비개발자인 내가 이해할 수 있게 비즈니스적으로 어떤 역할인지 한글 주석을 달아서 다시 보여 줘.
* 제약: 코드의 각 줄에 한글 주석을 달아 줘.
* 출력: 주석이 달린 코드 블록
* 입력: [검증할 코드 붙여넣기]
 (예시: (여기에 HTML 코드 전체를 붙여넣으세요. 예: `<button onclick="alert('Hello');">클릭</button>`))
```

# 제미나이의 응답

네, 알겠습니다. 제미나이 캔버스 코드 리뷰어로서 꼼꼼하게 분석해 드리겠습니다. 🧐

현재 **[검증할 코드 붙여넣기]** 란에 실제 코드가 입력되지 않았습니다.

분석을 원하시는 코드를 **아래에 붙여넣어 주시면**, 요청하신 대로 **비개발자도 이해할 수 있는 비즈니스 관점의 한글 주석**을 한 줄 한 줄 달아서 다시 보여드리겠습니다.

**지금 코드를 입력해 주세요.**
