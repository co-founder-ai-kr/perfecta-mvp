# 민순의 요청

```text
* 역할: 너는 제미나이 캔버스 코드를 검수하는 꼼꼼한 코드 리뷰어야.
* 상황: 작성된 코드가 내가 의도한 비즈니스 로직대로 정확히 구현되었는지 검증하고 싶어.
* 과업: 이 코드를 한 줄 한 줄 분석해서, 비개발자인 내가 이해할 수 있게 비즈니스적으로 어떤 역할인지 한글 주석을 달아서 다시 보여 줘.
* 제약: 코드의 각 줄에 한글 주석을 달아 줘.
* 출력: 주석이 달린 코드 블록
* 입력: [검증할 코드 붙여넣기]
 (예시: `<button onclick="alert('Hello');">클릭</button>`)
```

# 제미나이의 응답

네, 알겠습니다. 꼼꼼한 코드 리뷰어로서 작성된 코드가 의도하신 비즈니스 로직과 일치하는지 비개발자의 시선에서 이해하기 쉽게 분석해 드리겠습니다.

**검증할 코드를 붙여넣어 주시면 바로 작업을 시작하겠습니다.**

우선, 예시로 들어주신 코드를 바탕으로 제가 어떻게 주석을 달아드릴지 미리 보여드리겠습니다.

```html
<!-- [UI 구성] 화면에 사용자가 누를 수 있는 버튼 요소를 생성합니다. -->
<button 
    onclick="alert('Hello');" <!-- [동작 정의] 사용자가 이 버튼을 클릭하면, 브라우저 알림창을 통해 'Hello'라는 환영 메시지를 즉시 보여줍니다. -->
>
    클릭 <!-- [화면 표시] 버튼 위에 '클릭'이라는 글자를 표시하여 사용자가 무엇을 해야 할지 알려줍니다. -->
</button>
```

**이제 분석이 필요한 코드를 아래에 붙여넣어 주세요.**
