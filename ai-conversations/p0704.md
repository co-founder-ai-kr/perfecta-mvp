# 민순의 요청

```text
* 역할: 너는 CSS 스타일링 전문 디자이너야.
* 상황: 디자인 감각이 없어서 색 조합이 촌스러워. 특정 컨셉으로 세련되게 바꾸고 싶어.
* 과업: 입력된 스타일 키워드를 바탕으로 전체 디자인 컨셉을 잡아 줘.
* 제약:
  - 색상과 폰트, 버튼 스타일을 해당 컨셉에 맞게 코드로 구현해 줘.
* 출력: CSS 코드
* 입력: [원하는 스타일 키워드]
 (예시: 레트로 스타일 - 머스타드 옐로우, 딥 그린, 명조체, 타자기 같은 버튼)
```

# 제미나이의 응답

제시해주신 예시 키워드 **[레트로 스타일 - 머스타드 옐로우, 딥 그린, 명조체, 타자기 같은 버튼]**을 바탕으로 디자인 컨셉을 잡고 CSS 코드를 작성해 드립니다.

이 컨셉은 **"Modern Nostalgia (모던 노스탤지어)"** 테마로 정의했습니다. 따뜻하고 아날로그적인 감성을 주면서도 웹에서 가독성을 해치지 않도록 구성했습니다.

### 🎨 디자인 컨셉 상세
*   **색상 팔레트**: 쨍한 원색 대신 채도를 살짝 낮춘 머스타드와 깊이감 있는 딥 그린을 사용하여 눈이 편안하면서도 고풍스러운 느낌을 줍니다. 배경은 순백색이 아닌 크림색을 사용하여 낡은 종이 질감을 연상시킵니다.
*   **타이포그래피**: 본문은 가독성이 좋은 명조체(Serif)를 사용하고, 포인트가 되는 버튼이나 제목에는 타자기 서체(Monospace)를 사용하여 아날로그 기계의 느낌을 더했습니다.
*   **버튼 스타일**: 플랫(Flat)한 디자인 대신, 뚜렷한 테두리와 그림자(Hard Shadow)를 사용하여 타자기 키를 누르는 듯한 물리적인 입체감을 부여했습니다.

---

### 💻 CSS Code

```css
/* 1. 폰트 임포트 (구글 폰트: Noto Serif KR, Courier Prime) */
@import url('https://fonts.googleapis.com/css2?family=Courier+Prime:wght@400;700&family=Noto+Serif+KR:wght@400;700&display=swap');

/* 2. 디자인 토큰 (변수 설정) */
:root {
    /* Color Palette */
    --retro-mustard: #E1AD01;       /* 따뜻한 머스타드 옐로우 */
    --retro-green: #1A4D2E;         /* 차분하고 깊은 딥 그린 */
    --retro-cream: #FDFBF7;         /* 낡은 종이 느낌의 크림색 배경 */
    --retro-text: #2C2C2C;          /* 완전한 검정보다 부드러운 짙은 회색 */
    --retro-border: #1A4D2E;        /* 테두리는 딥 그린으로 통일 */

    /* Typography */
    --font-main: 'Noto Serif KR', serif;       /* 명조체 */
    --font-point: 'Courier Prime', monospace;  /* 타자기체 */
}

/* 3. 기본 베이스 스타일 */
body {
    background-color: var(--retro-cream);
    color: var(--retro-text);
    font-family: var(--font-main);
    line-height: 1.6;
    letter-spacing: -0.02em;
}

h1, h2, h3 {
    color: var(--retro-green);
    font-weight: 700;
    margin-bottom: 1rem;
}

/* 4. 타자기 스타일 버튼 (핵심 디자인) */
.btn-retro {
    /* 레이아웃 및 폰트 */
    display: inline-block;
    padding: 12px 24px;
    font-family: var(--font-point); /* 타자기체 적용 */
    font-size: 1rem;
    font-weight: 700;
    text-decoration: none;
    cursor: pointer;
    
    /* 색상 및 테두리 */
    background-color: var(--retro-mustard);
    color: var(--retro-green);
    border: 2px solid var(--retro-border);
    border-radius: 4px; /* 아주 살짝 둥글게 */
    
    /* 레트로한 입체감 (Hard Shadow) */
    box-shadow: 4px 4px 0px var(--retro-green);
    
    /* 애니메이션 */
    transition: all 0.2s ease;
}

/* 버튼 호버 효과 */
.btn-retro:hover {
    background-color: #FFC82C; /* 호버 시 조금 더 밝게 */
    transform: translateY(-2px); /* 살짝 떠오르는 느낌 */
    box-shadow: 6px 6px 0px var(--retro-green); /* 그림자 길어짐 */
}

/* 버튼 클릭(Active) 효과 - 타자기 누르는 느낌 */
.btn-retro:active {
    transform: translateY(4px); /* 눌리는 느낌 */
    box-shadow: 0px 0px 0px var(--retro-green); /* 그림자 사라짐 */
    background-color: #D4A000; /* 눌렸을 때 약간 어둡게 */
}

/* 5. 카드/컨테이너 스타일 (추가 요소) */
.card-retro {
    background-color: #FFFFFF;
    border: 1px solid var(--retro-border);
    padding: 2rem;
    box-shadow: 8px 8px 0px rgba(26, 77, 46, 0.2); /* 반투명한 딥그린 그림자 */
}
```

### 📝 HTML 적용 예시

```html
<div class="card-retro">
    <h1>레트로 디자인 컨셉</h1>
    <p>
        이곳은 명조체가 적용된 본문 영역입니다. 
        오래된 서적을 읽는 듯한 차분한 느낌을 줍니다.
    </p>
    <br>
    <!-- 버튼 적용 -->
    <button class="btn-retro">Click Me</button>
</div>
```
