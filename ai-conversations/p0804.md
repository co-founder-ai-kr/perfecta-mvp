# 민순의 요청

```text
* 역할: 너는 구글 폼과 이메일 자동화 전문가야.
* 상황: 1인 창업가가 별도 조회 페이지 없이 이메일로 주문 내역을 자동 발송하여 고객을 안심시키고 싶어 해.
* 과업: 
  1. 구글 폼 제출 시 응답 사본을 자동으로 이메일로 보내는 설정 방법을 알려 줘.
  2. 더 예쁜 ‘주문 확인서’ 양식을 위한 도구 키워드를 추천해 줘.
* 제약: 구글 폼 설정 탭의 메뉴 위치를 정확히 알려 줘.
* 출력: 
  1. 설정 단계 설명
  2. 추천 도구 키워드
* 입력: [구글 폼의 목적]
 (예시: 주문 접수용)
```

# 제미나이의 응답

반갑습니다. 1인 창업가님을 위한 **구글 폼 및 이메일 자동화 전문가**입니다.

고객이 주문을 마친 후 "내 주문이 제대로 들어갔나?" 불안해하지 않도록, 즉시 주문 내역을 발송해 주는 것은 신뢰도 상승에 매우 중요한 포인트입니다. 별도의 개발 없이 구글 폼 기본 기능만으로 이를 해결하는 방법과, 더 나아가 브랜딩을 입힐 수 있는 도구 키워드를 정리해 드립니다.

입력해주신 **[주문 접수용]** 구글 폼을 기준으로 설명해 드리겠습니다.

---

### 1. 구글 폼 기본 설정: 응답 사본 자동 발송 방법

구글 폼 자체 기능을 활용하여, 고객이 작성한 내용을 그대로 이메일로 보내주는 가장 기초적이고 확실한 방법입니다.

**[설정 단계]**

1.  구글 폼 편집 화면 상단 중앙의 **[설정]** 탭을 클릭합니다.
2.  메뉴 중 **[응답]** 섹션의 화살표(∨)를 눌러 메뉴를 펼칩니다.
3.  **[이메일 주소 수집]** 항목을 찾습니다.
    *   기본값인 '수집 안함'을 클릭하여 **'응답자 입력'** (또는 '확인됨')으로 변경합니다.
    *   *팁: '응답자 입력'을 선택해야 고객이 로그인 없이도 이메일을 적고 주문할 수 있습니다.*
4.  바로 아래 **[응답 사본 전송]** 항목을 찾습니다.
5.  기본값인 '해제'를 클릭하여 **[항상]**으로 변경합니다.
    *   *'요청 시'로 설정하면 고객이 체크박스를 눌러야만 발송되므로, 무조건 발송되는 '항상'을 추천합니다.*

**결과:** 이제 고객이 폼을 제출하면, "Google Forms <forms-receipts-noreply@google.com>" 명의로 고객이 작성한 주문 내역 전체가 즉시 발송됩니다.

---

### 2. 더 예쁜 '주문 확인서'를 위한 도구 키워드 추천

구글 폼 기본 이메일은 디자인이 투박하고 단순 텍스트/표 형태로만 전송됩니다. 우리 브랜드 로고가 박힌 깔끔한 **견적서(Invoice)나 영수증 형태**로 보내고 싶다면 아래 도구들을 활용해 보세요.

**추천 도구 키워드 (구글 워크스페이스 마켓플레이스 검색용)**

1.  **Form Publisher (폼 퍼블리셔)**
    *   **특징:** 가장 대중적인 도구입니다. 구글 닥스(Docs)나 스프레드시트로 미리 예쁜 주문서 양식을 만들어두면, 고객 주문 내용이 그 양식에 자동으로 채워져 **PDF 파일**로 변환되어 이메일에 첨부됩니다.
    *   **추천 대상:** 종이 주문서 같은 깔끔한 PDF 파일을 고객에게 보내주고 싶을 때.

2.  **Autocrat (오토크랫)**
    *   **특징:** Form Publisher와 유사하지만 무료 기능 범위가 넓고 세부 설정이 강력합니다. 구글 시트와 연동하여 대량의 주문을 처리하고 맞춤형 이메일을 보내기에 적합합니다.
    *   **추천 대상:** 비용을 아끼면서 조금 더 디테일한 자동화 설정을 하고 싶을 때.

3.  **Zapier (재피어)**
    *   **특징:** 구글 폼과 'Gmail'을 직접 연결하는 자동화 도구입니다. PDF 첨부보다는 **이메일 본문 자체**를 예쁘게(HTML) 꾸며서 "OOO님, 주문해 주셔서 감사합니다."라는 친근한 멘트와 함께 보내고 싶을 때 사용합니다.
    *   **추천 대상:** 파일 첨부보다는 이메일 본문의 디자인과 텍스트 톤앤매너가 중요할 때.

**전문가의 조언:**
처음에는 **1번(기본 설정)**으로 시작하여 시스템이 잘 돌아가는지 확인하신 후, 주문량이 늘어나면 **Form Publisher** 등을 도입하여 브랜드의 전문성을 높이는 단계를 추천합니다.
