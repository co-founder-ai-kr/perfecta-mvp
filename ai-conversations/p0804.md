# 민순의 요청

```text
* 역할: 너는 구글 폼과 이메일 자동화 전문가야.
* 상황: 별도 조회 페이지 없이 이메일로 주문 내역을 자동 발송하여 고객을 안심시키고 싶어.
* 과업: 
  1. 구글 폼 제출 시 응답 사본을 자동으로 이메일로 보내는 설정 방법을 알려 줘.
  2. 더 예쁜 ‘주문 확인서’ 양식을 위한 도구 키워드를 추천해 줘.
* 제약: 구글 폼 설정 탭의 메뉴 위치를 정확히 알려 줘.
* 출력: 
  1. 설정 단계 설명
  2. 추천 도구 키워드
* 입력: [구글 폼의 목적]
 (예시: 주문 접수용)
```

# 제미나이의 응답

입력해주신 **[주문 접수용]** 구글 폼의 목적에 맞춰, 고객이 주문 직후 안심할 수 있도록 **자동 응답 이메일 설정법**과 **고급 도구 키워드**를 정리해 드립니다.

---

### 1. 구글 폼 기본 설정: 응답 사본 자동 발송 (단계별 설명)

구글 폼 자체 기능을 활용하여, 고객이 작성한 내용을 그대로 이메일로 보내주는 가장 기초적이고 확실한 방법입니다.

1.  구글 폼 편집 화면 상단 중앙의 **[설정]** 탭을 클릭합니다.
2.  메뉴 목록 중 **[응답]** 섹션의 화살표(∨)를 눌러 세부 메뉴를 펼칩니다.
3.  **[이메일 주소 수집]** 항목을 확인합니다.
    *   이 항목이 '수집 안함'으로 되어 있다면, **'응답자 입력'** 또는 **'확인됨'**으로 변경해야 합니다. (이메일 주소가 있어야 발송이 가능하기 때문입니다.)
4.  바로 아래에 있는 **[응답자에게 응답 사본 전송]** 항목을 찾습니다.
5.  기본값인 '해제'를 클릭하여 **[항상]**으로 변경합니다.
    *   *'요청 시'*로 설정하면 고객이 체크박스를 눌러야만 발송되므로, 주문 확인용이라면 **'항상'**으로 설정하는 것이 필수입니다.

> **💡 결과:** 이제 고객이 폼을 제출(주문 완료)하면, 입력한 이메일 주소로 "주문 내역(작성한 답변)"이 포함된 구글 기본 양식의 이메일이 즉시 발송됩니다.

---

### 2. 더 예쁜 ‘주문 확인서’를 위한 도구 키워드 추천

구글 폼 기본 이메일은 디자인이 투박합니다. 로고를 넣거나, "OOO님 주문 감사합니다"와 같이 정중하고 예쁜 디자인의 이메일(또는 PDF 영수증)을 보내고 싶다면 아래 키워드의 도구(부가기능)를 사용해야 합니다.

**추천 도구 키워드 (구글 워크스페이스 마켓플레이스 검색용)**

*   **Autocrat (오토크랫):**
    *   **특징:** 가장 유명한 무료(일부 유료) 도구입니다. 구글 닥스(Docs)나 슬라이드(Slides)로 예쁜 주문서 양식을 미리 만들어두면, 주문이 들어올 때마다 내용을 갈아 끼워 **PDF 파일로 변환해 이메일에 첨부**해 줍니다.
*   **Form Publisher (폼 퍼블리셔):**
    *   **특징:** Autocrat과 유사하지만 설정이 조금 더 직관적입니다. 주문 내역을 깔끔한 문서 형태로 만들어 결재 서류나 영수증처럼 보내기에 적합합니다.
*   **Email Notifications for Google Forms:**
    *   **특징:** 첨부파일 없이 **이메일 본문 자체를 예쁘게(HTML)** 꾸미고 싶을 때 강력합니다. 고객의 이름, 주문 상품 등을 이메일 본문에 자연스럽게 녹여낼 수 있습니다.

**전문가 팁:**
처음에는 **1번(기본 설정)**을 먼저 적용하여 시스템을 안정화한 뒤, 브랜드 이미지가 중요해지는 시점에 **Autocrat** 등을 도입하여 브랜딩 된 PDF 주문 확인서를 발송하는 것을 추천합니다.
