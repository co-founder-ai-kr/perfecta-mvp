# 민순의 요청

```text
* 역할: 너는 제미나이 캔버스 환경에 정통한 베테랑 소프트웨어 아키텍트야.
* 상황: 복잡한 설정 없이 서비스를 만들고 싶어.
* 과업: 유지 보수하기 쉽고, 제미나이 캔버스에서 별도 설정 없이 바로 실행되는 단순하고 효과적인 기술 조합 추천해 줘.
* 제약:
  - 설치 과정이 아예 없었으면 좋겠어.
  - 디자인, 기능, 내용이 파일 하나에 다 들어가는 방식으로 짜 줘.
  - 코드에는 개발 용어 말고 비즈니스 역할을 설명하는 주석 달아 줘.
* 출력: 추천 기술 스택 목록
* 입력: [만들고 싶은 서비스 개요]
 (예시: 선물 추천 서비스)
```

# 제미나이의 응답

반갑습니다. 제미나이 캔버스 환경의 특성과 '설치 없는 즉시 실행'이라는 제약 조건을 고려했을 때, 가장 강력하고 유지보수가 용이한 기술 조합을 제안해 드립니다.

복잡한 프레임워크(React, Vue 등)는 빌드 과정이나 외부 라이브러리 로딩이 필요하여 캔버스 환경에서 가끔 충돌을 일으킬 수 있습니다. 따라서 **브라우저가 태생적으로 이해하는 표준 기술**만 사용하는 것이 가장 안전하고 빠릅니다.

### 🏗️ 추천 기술 스택: [Vanilla Single-File Architecture]

이 조합은 별도의 설치가 전혀 필요 없으며, 텍스트 파일 하나만 있으면 어디서든(제미나이 캔버스, 브라우저, 모바일 등) 즉시 실행됩니다.

1.  **구조 (HTML5):** 건물의 뼈대와 인테리어 구획을 담당합니다.
2.  **스타일 (CSS3):** 고객이 보게 될 디자인과 브랜드 컬러를 입힙니다.
3.  **로직 (Vanilla JavaScript):** 점원이 어떻게 응대하고 계산할지 업무 매뉴얼을 정의합니다.

---

### 📋 아키텍처 템플릿 (복사해서 바로 사용하세요)

아래 코드는 **[만들고 싶은 서비스 개요]**만 입력하면 바로 작동하도록 설계된 뼈대입니다. 개발 용어 대신 비즈니스 역할을 설명하는 주석을 달았습니다.

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>서비스 이름</title>
    <style>
        /* [디자인 부서] : 고객이 보는 화면을 꾸미는 곳입니다. */
        
        :root {
            /* 브랜드 컬러 정의 */
            --main-color: #4A90E2; 
            --bg-color: #f4f6f8;
            --card-bg: #ffffff;
        }

        body {
            font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Helvetica, Arial, sans-serif;
            background-color: var(--bg-color);
            margin: 0;
            padding: 20px;
            display: flex;
            justify-content: center;
            align-items: center;
            min-height: 100vh;
        }

        /* 서비스 전체를 감싸는 매장 외관 */
        .app-container {
            background-color: var(--card-bg);
            width: 100%;
            max-width: 400px; /* 모바일 친화적 너비 */
            padding: 30px;
            border-radius: 16px;
            box-shadow: 0 4px 20px rgba(0,0,0,0.1);
        }

        h1 {
            text-align: center;
            color: #333;
            margin-bottom: 20px;
        }

        /* 입력 창 디자인 */
        .input-group {
            display: flex;
            gap: 10px;
            margin-bottom: 20px;
        }

        input {
            flex-grow: 1;
            padding: 12px;
            border: 1px solid #ddd;
            border-radius: 8px;
            outline: none;
        }

        /* 실행 버튼 디자인 */
        button {
            background-color: var(--main-color);
            color: white;
            border: none;
            padding: 12px 20px;
            border-radius: 8px;
            cursor: pointer;
            font-weight: bold;
            transition: 0.2s;
        }

        button:hover {
            opacity: 0.9;
        }

        /* 결과가 보여질 진열대 디자인 */
        #result-area {
            margin-top: 20px;
            padding: 15px;
            background-color: #f8f9fa;
            border-radius: 8px;
            min-height: 50px;
            color: #555;
            line-height: 1.6;
        }
    </style>
</head>
<body>

    <!-- [매장 구조] : 실제 화면에 배치되는 요소들입니다. -->
    <div class="app-container">
        <!-- 간판 -->
        <h1 id="app-title">서비스 제목</h1>

        <!-- 고객 요청 접수 창구 -->
        <div class="input-group">
            <input type="text" id="user-input" placeholder="원하는 내용을 입력하세요">
            <button id="action-btn">실행</button>
        </div>

        <!-- 결과 제공 창구 -->
        <div id="result-area">
            여기에 결과가 나타납니다.
        </div>
    </div>

    <script>
        /* [업무 매뉴얼] : 서비스가 실제로 어떻게 동작할지 규칙을 정하는 곳입니다. */

        // 1. 업무에 필요한 도구와 장소를 미리 파악합니다.
        const inputField = document.getElementById('user-input'); // 고객의 주문서
        const actionButton = document.getElementById('action-btn'); // 주문 접수 버튼
        const resultDisplay = document.getElementById('result-area'); // 결과물을 보여줄 전광판

        // 2. 서비스의 핵심 데이터를 준비합니다 (창고).
        const serviceData = {
            title: "나만의 서비스",
            defaultMessage: "무엇을 도와드릴까요?"
        };

        // 3. 초기 세팅 업무 (가게 문 열 때 하는 일)
        function init() {
            document.getElementById('app-title').innerText = serviceData.title;
            resultDisplay.innerText = serviceData.defaultMessage;
        }

        // 4. 핵심 비즈니스 로직 (실제 서비스가 제공하는 가치)
        function processRequest() {
            const customerRequest = inputField.value; // 고객이 적은 내용 가져오기

            if (!customerRequest) {
                alert("내용을 입력해주세요!"); // 내용이 없으면 안내
                return;
            }

            // --- [여기서부터 서비스의 핵심 기능을 구현합니다] ---
            // 예시: 입력된 텍스트를 가공해서 보여주기
            const result = `처리 결과: "${customerRequest}"에 대한 분석이 완료되었습니다.`;
            // ------------------------------------------------

            // 결과 전광판에 출력
            resultDisplay.innerText = result;
            
            // 입력창 비우기 (다음 주문을 위해)
            inputField.value = '';
        }

        // 5. 고객 반응 대기 (버튼을 눌렀을 때, 엔터키를 쳤을 때)
        actionButton.addEventListener('click', processRequest);
        
        inputField.addEventListener('keypress', function(e) {
            if (e.key === 'Enter') {
                processRequest();
            }
        });

        // 서비스 시작
        init();

    </script>
</body>
</html>
```
