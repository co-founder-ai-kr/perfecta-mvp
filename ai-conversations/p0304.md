# 민순의 요청

```text
* 역할: 너는 제미나이 캔버스 환경에 정통한 베테랑 소프트웨어 아키텍트야.
* 상황: 복잡한 설정 없이 서비스를 만들고 싶어.
* 과업: 'HTML/CSS/JS'를 별도 파일로 나누지 않고 'index.html' 파일 하나에 모두 담는 '단일 파일 구조'로 기술 스택을 확정했어. 이 구조를 가장 효율적으로 짤 수 있는 기본 뼈대 코드를 작성해 줘.
* 제약:
  - 설치 과정이 아예 없었으면 좋겠어. (Node.js, React, Build 도구 절대 사용 금지)
  - 오직 순수 HTML, CSS, Vanilla JavaScript만 사용해서 파일 하나에 다 들어가는 방식으로 짜 줘.

* 출력: 추천 기술 스택 목록
* 입력: [만들고 싶은 서비스 개요]
 (예시: 선물 추천 서비스)
```

# 제미나이의 응답

반갑습니다. 제미나이 캔버스 환경에서 가장 빠르고 효율적으로 MVP(Minimum Viable Product)를 검증할 수 있는 아키텍처를 제안해 드리겠습니다.

요청하신 **'단일 파일 구조(Single File Structure)'**는 복잡한 빌드 도구 없이 브라우저만 있으면 즉시 실행 가능한, 초기 아이디어 검증에 최적화된 방식입니다.

입력해주신 예시인 **[선물 추천 서비스]**를 기준으로, 바로 복사해서 사용할 수 있는 뼈대 코드를 작성했습니다.

---

### 1. 추천 기술 스택 (Zero-Config Stack)

이 스택은 '설치'가 필요 없으며, 파일 저장 후 브라우저에서 열기만 하면 작동합니다.

*   **구조 (Structure):** HTML5 (시맨틱 태그 활용)
*   **스타일 (Style):** CSS3 (Flexbox/Grid 레이아웃, CSS Variables 활용)
*   **로직 (Logic):** Vanilla JavaScript (ES6+ 문법, 별도 라이브러리 없음)
*   **데이터 (Data):** 하드코딩된 JSON 객체 (초기 단계) 또는 `localStorage` (브라우저 저장소)
*   **배포 (Deployment):** GitHub Pages 또는 Netlify Drop (HTML 파일 드래그 앤 드롭으로 배포 가능)

---

### 2. 단일 파일 코드 (index.html)

아래 코드를 복사하여 `index.html`로 저장한 뒤 브라우저(Chrome, Edge 등)에서 여세요.
**주석은 개발 용어가 아닌, 비즈니스 관점에서의 역할로 작성되었습니다.**

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>선물 추천 서비스 MVP</title>
    <style>
        /* [인테리어 담당] 서비스의 전반적인 디자인과 배치를 담당합니다. */
        :root {
            --primary-color: #6200ea; /* 브랜드 메인 컬러 */
            --bg-color: #f5f5f7;      /* 매장 배경색 */
            --card-bg: #ffffff;       /* 상품 진열대 배경색 */
        }

        body {
            font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Helvetica, Arial, sans-serif;
            background-color: var(--bg-color);
            margin: 0;
            display: flex;
            justify-content: center;
            min-height: 100vh;
        }

        /* 매장(앱)의 메인 컨테이너 */
        #app-container {
            width: 100%;
            max-width: 600px; /* 모바일 친화적인 폭 */
            background-color: var(--card-bg);
            padding: 2rem;
            box-shadow: 0 4px 20px rgba(0,0,0,0.1);
            display: flex;
            flex-direction: column;
            gap: 20px;
        }

        h1 { text-align: center; color: #333; }
        
        /* 버튼 스타일 */
        .action-btn {
            background-color: var(--primary-color);
            color: white;
            border: none;
            padding: 15px;
            border-radius: 10px;
            font-size: 1rem;
            cursor: pointer;
            transition: opacity 0.2s;
        }
        .action-btn:hover { opacity: 0.9; }

        /* 결과 카드 스타일 */
        .gift-card {
            border: 1px solid #eee;
            padding: 15px;
            border-radius: 8px;
            margin-bottom: 10px;
            background: #fafafa;
        }
        .gift-name { font-weight: bold; font-size: 1.1rem; }
        .gift-desc { color: #666; font-size: 0.9rem; margin-top: 5px; }

        /* 화면 전환용 유틸리티 */
        .hidden { display: none !important; }
    </style>
</head>
<body>

    <!-- [매장 입구] 고객이 서비스를 이용하는 메인 공간입니다. -->
    <div id="app-container">
        
        <!-- 1. 시작 화면 (랜딩 페이지) -->
        <section id="view-landing">
            <h1>🎁 맞춤 선물 추천</h1>
            <p style="text-align: center; color: #666;">
                누구를 위한 선물인가요?<br>
                몇 가지 질문에 답하면 딱 맞는 선물을 찾아드려요.
            </p>
            <button class="action-btn" style="width: 100%;" onclick="manager.startConsultation()">
                추천 받기 시작
            </button>
        </section>

        <!-- 2. 질문 화면 (상담 구역) -->
        <section id="view-question" class="hidden">
            <h2 id="question-text">질문 내용이 여기에 표시됩니다.</h2>
            <div id="options-container" style="display: flex; flex-direction: column; gap: 10px;">
                <!-- 선택지 버튼들이 여기에 동적으로 생성됩니다 -->
            </div>
        </section>

        <!-- 3. 결과 화면 (상품 진열대) -->
        <section id="view-result" class="hidden">
            <h2>추천 결과</h2>
            <div id="result-list">
                <!-- 추천된 상품들이 여기에 진열됩니다 -->
            </div>
            <button class="action-btn" style="width: 100%; margin-top: 20px;" onclick="manager.resetService()">
                다시 추천 받기
            </button>
        </section>

    </div>

    <script>
        /**
         * [창고 관리자]
         * 우리 가게가 보유한 선물 목록(데이터)입니다.
         * 실제 서비스에서는 서버에서 가져오겠지만, 지금은 장부에 적어둡니다.
         */
        const inventory = [
            { id: 1, name: "고급 만년필", type: "business", price: "high", desc: "성공을 기원하는 품격 있는 선물" },
            { id: 2, name: "무선 이어폰", type: "tech", price: "mid", desc: "출퇴근길 필수 아이템" },
            { id: 3, name: "아로마 캔들", type: "healing", price: "low", desc: "지친 하루를 위로하는 향기" },
            { id: 4, name: "게이밍 키보드", type: "tech", price: "high", desc: "게이머가 가장 받고 싶어하는 장비" },
            { id: 5, name: "비타민 세트", type: "health", price: "mid", desc: "건강을 챙겨주는 따뜻한 마음" }
        ];

        /**
         * [상담 매뉴얼]
         * 고객에게 물어볼 질문 리스트입니다.
         */
        const consultationManual = [
            {
                id: 1,
                text: "선물을 받는 분은 어떤 스타일인가요?",
                options: [
                    { label: "일잘러 직장인", value: "business" },
                    { label: "얼리어답터/테크", value: "tech" },
                    { label: "집순이/집돌이", value: "healing" },
                    { label: "건강이 최고", value: "health" }
                ]
            },
            // 필요하다면 질문을 더 추가할 수 있습니다.
        ];

        /**
         * [총괄 매니저]
         * 서비스의 전체 흐름을 제어하고 고객 응대를 담당하는 핵심 로직입니다.
         */
        const manager = {
            // 고객의 답변을 임시로 적어두는 메모장
            customerPreferences: {},

            // [행동] 상담 시작하기
            startConsultation: () => {
                ui.switchView('view-question');
                manager.askQuestion(0); // 첫 번째 질문 던지기
            },

            // [행동] 질문 보여주기
            askQuestion: (index) => {
                const question = consultationManual[index];
                const container = document.getElementById('options-container');
                
                // 질문 텍스트 변경
                document.getElementById('question-text').innerText = question.text;
                
                // 기존 선택지 지우고 새로 만들기
                container.innerHTML = '';
                
                question.options.forEach(opt => {
                    const btn = document.createElement('button');
                    btn.className = 'action-btn';
                    btn.style.backgroundColor = '#eee'; // 선택지는 회색으로
                    btn.style.color = '#333';
                    btn.innerText = opt.label;
                    
                    // 버튼 클릭 시 답변 기록 후 결과 분석
                    btn.onclick = () => {
                        manager.customerPreferences.type = opt.value; // 답변 기록 (단순화됨)
                        manager.analyzeAndRecommend(); // 바로 추천 (질문이 1개라 가정)
                    };
                    container.appendChild(btn);
                });
            },

            // [행동] 답변 분석 및 상품 추천
            analyzeAndRecommend: () => {
                // 고객이 선호하는 타입과 일치하는 상품을 창고에서 찾습니다.
                const recommendedItems = inventory.filter(item => 
                    item.type === manager.customerPreferences.type
                );

                ui.displayResults(recommendedItems);
                ui.switchView('view-result');
            },

            // [행동] 처음으로 돌아가기
            resetService: () => {
                manager.customerPreferences = {};
                ui.switchView('view-landing');
            }
        };

        /**
         * [인테리어 시공팀]
         * 화면을 실제로 보여주거나 숨기는 등 눈에 보이는 작업을 처리합니다.
         */
        const ui = {
            // 화면 전환 (특정 구역만 보여주고 나머지는 가림)
            switchView: (viewId) => {
                ['view-landing', 'view-question', 'view-result'].forEach(id => {
                    document.getElementById(id).classList.add('hidden');
                });
                document.getElementById(viewId).classList.remove('hidden');
            },

            // 결과 목록을 화면에 그리기
            displayResults: (items) => {
                const listContainer = document.getElementById('result-list');
                listContainer.innerHTML = '';

                if (items.length === 0) {
                    listContainer.innerHTML = '<p>조건에 맞는 상품이 없네요 😢</p>';
                    return;
                }

                items.forEach(item => {
                    const card = document.createElement('div');
                    card.className = 'gift-card';
                    card.innerHTML = `
                        <div class="gift-name">${item.name}</div>
                        <div class="gift-desc">${item.desc}</div>
                    `;
                    listContainer.appendChild(card);
                });
            }
        };
    </script>
</body>
</html>
```

### 3. 아키텍트의 조언 (사용 방법)

1.  **데이터 수정:** `inventory` 배열(창고)에 있는 데이터를 실제 기획하신 선물 목록으로 바꿔주세요.
2.  **질문 로직:** `consultationManual` 배열(상담 매뉴얼)을 수정하여 질문을 변경할 수 있습니다.
3.  **디자인:** `<style>` 태그 내의 `root` 변수(`--primary-color`)만 변경해도 앱의 분위기가 확 바뀝니다.

이 구조는 파일 하나로 모든 기능이 동작하므로, 제미나이 캔버스에서 코드를 수정하고 바로바로 결과를 확인하기에 가장 적합합니다.
