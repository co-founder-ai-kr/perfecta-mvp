# 민순의 요청

```text
* 역할: 너는 내 옆에 앉은 친절한 네틀리파이(Netlify) 배포 가이드야.
* 상황: 복잡한 명령어 없이 파일을 끌어다 놓는 방식으로 내 웹 사이트를 올리고 싶어.
* 과업: 
  - 'Netlify' 로그인 후, 파일을 드래그 앤 드롭할 수 있는 화면으로 바로 이동하는 직링크(Direct Link)를 알려 줘.
  - 만약 직링크가 없다면, 로그인 직후 화면에서 'Add new site' > 'Deploy manually' 버튼을 찾아가는 최단 경로를 알려 줘.
  - 현재 시점의 네틀리파이 UI를 기준으로 설명해 줘.
* 출력: 단계별 실행 가이드
```

# 제미나이의 응답

안녕! 옆자리에 앉은 네틀리파이 가이드야. 복잡한 건 딱 질색이지? 터미널이나 코드 입력 없이 마우스만으로 1분 안에 끝낼 수 있게 도와줄게.

네틀리파이는 사용자마다 고유한 '팀 ID'가 있어서 **모든 사람에게 통하는 '드래그 앤 드롭 화면 직링크'는 아쉽게도 없어.** 하지만 로그인만 하면 딱 **3번의 클릭**으로 그 화면에 갈 수 있는 가장 빠른 지름길을 알려줄게.

지금 바로 따라와 봐!

---

### 🚀 1단계: 로그인 및 시작점 이동
가장 먼저 아래 링크를 눌러서 로그인을 해줘. 로그인이 되어 있다면 바로 메인 대시보드로 이동할 거야.

*   **바로 가기:** **[https://app.netlify.com](https://app.netlify.com)**

---

### 🖱️ 2단계: 'Deploy manually' 버튼 찾기 (최단 경로)
로그인 후 화면이 나왔지? 이제 내 손가락을 잘 봐. 딱 여기만 누르면 돼.

1.  화면 왼쪽 사이드바 메뉴에서 **[Sites]**를 클릭해. (이미 눌려있다면 패스!)
2.  화면 오른쪽 위에 있는 **[Add new site]** 버튼을 찾았어?
    *   ⚠️ **주의:** 버튼을 바로 누르지 말고, 버튼 **오른쪽 끝에 있는 작은 화살표(▼)**를 눌러야 해.
3.  아래로 열리는 메뉴 중에서 **[Deploy manually]**를 클릭해.

---

### 📂 3단계: 파일 끌어다 놓기
이제 화면 중앙에 점선으로 된 박스가 보일 거야. **"Drag and drop your site output folder here"**라고 쓰여 있지?

*   준비한 웹 사이트 **폴더 전체**를 마우스로 잡아서 저 박스 안에 **툭** 던져 놓으면 끝!

---

### 💡 잠깐! 실패하지 않는 꿀팁
*   **파일 말고 폴더:** `index.html` 파일 하나만 넣으면 안 돼. 그 파일이 들어있는 **'폴더 자체'**를 끌어다 놔야 해.
*   **업로드 중:** 폴더를 놓으면 원형 로딩 바가 돌아갈 거야. "Published"라는 초록색 글씨가 뜨면 배포 성공이야!

자, 이제 드래그해 볼까? 잘 안 되면 언제든 다시 물어봐!
