# 민순의 요청

```text
* 역할: 너는 데이터 전처리 전문가야.
* 상황: 구글 시트의 날짜와 금액 데이터 형식이 텍스트와 숫자로 뒤섞여 있어 분석 오류가 나고 있어.
* 과업: 이걸 통일하는 메뉴 위치와 방법을 알려 줘.
* 제약:
  - 코딩 말고, 상단 메뉴의 버튼을 누르는 순서대로 설명할 것.
  - 단계별로 설명해 줘.
```

# 제미나이의 응답

반갑습니다. 데이터 전처리 전문가입니다.

구글 시트에서 날짜와 숫자가 텍스트로 인식되면 피벗 테이블이나 차트, 수식 계산에서 오류가 발생하죠. 코딩 없이 **상단 메뉴 버튼**만으로 이 문제를 가장 빠르고 정확하게 해결하는 방법을 단계별로 알려드리겠습니다.

---

### 1. 날짜 데이터 통일하기 (텍스트 → 날짜 형식)

날짜가 '2023.10.25' 처럼 보이지만 실제로는 텍스트로 인식되어 정렬이 안 될 때 사용합니다.

**[방법 A: 표준 서식 적용]**
1.  날짜가 적힌 **열(Column) 전체를 선택**합니다 (예: A열 헤더 클릭).
2.  상단 메뉴에서 **[서식]**을 클릭합니다.
3.  **[숫자]**에 마우스를 올립니다.
4.  나오는 목록에서 **[날짜]**를 클릭합니다.

**[방법 B: 강제 변환 (가장 확실한 방법)]**
*위 방법으로도 변환이 안 되는 '고집 센 텍스트'가 있을 때 사용하세요.*
1.  날짜 열을 선택합니다.
2.  상단 메뉴에서 **[데이터]**를 클릭합니다.
3.  **[텍스트를 열로 분할]**을 클릭합니다.
4.  열 하단에 작은 메뉴창이 뜨면 **'구분선: 자동으로 감지'** 상태 그대로 둡니다. (아무것도 건드리지 않아도 시스템이 데이터를 다시 읽어들이며 날짜로 변환합니다.)

---

### 2. 금액 데이터 통일하기 (텍스트 → 숫자/통화 형식)

금액에 콤마(,)나 원화 표시(₩)가 텍스트로 섞여 있어 계산이 안 될 때 사용합니다.

**[단계 1: 공백 제거 (전처리 필수)]**
*숫자 앞뒤에 눈에 안 보이는 공백이 있으면 숫자로 변환되지 않습니다.*
1.  금액 열 전체를 선택합니다.
2.  상단 메뉴에서 **[데이터]**를 클릭합니다.
3.  **[데이터 정리]**에 마우스를 올립니다.
4.  **[공백 제거]**를 클릭합니다. (선택한 셀의 앞뒤 공백이 사라집니다.)

**[단계 2: 숫자 서식 적용]**
1.  금액 열이 선택된 상태에서 상단 메뉴 **[서식]**을 클릭합니다.
2.  **[숫자]**에 마우스를 올립니다.
3.  원하는 형식을 선택합니다:
    *   순수 숫자가 필요하면: **[숫자]** (1,000 형태)
    *   화폐 단위가 필요하면: **[통화]** (₩1,000 형태)

---

### 💡 전문가의 팁
만약 위 과정을 거쳤는데도 특정 셀이 여전히 왼쪽 정렬(텍스트)로 남아있다면, 해당 셀의 데이터 자체가 오타(예: 2023-02-30 같은 없는 날짜, 혹은 숫자 사이에 문자가 섞임)일 확률이 높습니다. 이 경우 해당 셀만 직접 수정해주시면 됩니다.

이제 순서대로 버튼을 눌러 데이터를 정리해 보세요!
