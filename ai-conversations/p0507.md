# 민순의 요청

```text
* 역할: 너는 친절한 구글 시트(Google Sheets) 활용 전문가야.
* 상황: 엑셀 함수를 전혀 모르는데, 마우스 클릭만으로 통계를 내고 싶어.
* 과업:
  1. 각 항목별로 몇 명이 선택했는지 숫자를 세고 싶어. 복잡한 수식 없이 메뉴에서 클릭만으로 표를 요약하는 기능(피벗 테이블 등) 사용법 안내.
  2. 내가 데이터를 선택하고 어디를 눌러야 하는지 단계별로 설명.
* 제약: 어려운 용어 쓰지 말고 "A를 끌어다가 B에 놓으세요"처럼 행동 중심으로 알려 줘.
* 출력: 따라 하기 쉬운 단계별 가이드(Step-by-step)
* 입력: [데이터 구조 설명]
 (예시: A열은 날짜, B열은 선택한 선물 종류)
```

# 제미나이의 응답

안녕하세요! 복잡한 함수 없이 마우스 클릭만으로 데이터를 요약하고 싶으시군요. 정말 잘 찾아오셨습니다.

구글 시트에는 **'피벗 테이블(Pivot Table)'**이라는 아주 강력한 기능이 있습니다. 이름은 어렵게 들리지만, 사실은 **"블록 쌓기"**처럼 원하는 항목을 끌어다 놓기만 하면 끝나는 아주 쉬운 기능이에요.

**"B열에 있는 선물 종류별로 몇 명이 선택했는지"** 세는 방법을 아주 쉽게 단계별로 알려드릴게요.

---

### 🖱️ 마우스만 사용하는 3단계 가이드

#### 1단계: 데이터 선택하고 메뉴 누르기
가장 먼저 컴퓨터에게 "이 데이터를 요약해 줘"라고 알려줘야 합니다.

1.  데이터가 들어있는 **A열과 B열의 알파벳 부분(맨 위)**을 마우스로 드래그해서 선택하세요. (데이터 전체가 파랗게 잡힙니다.)
2.  화면 맨 위 메뉴에서 **[삽입]**을 클릭하세요.
3.  나오는 메뉴 중 **[피벗 테이블]**을 클릭하세요.
4.  작은 창이 뜨면 아무것도 건드리지 말고 **[만들기]** 버튼을 누르세요.
    *   *결과: 화면 아래쪽에 '피벗 테이블 1'이라는 새로운 시트(탭)가 생깁니다.*

#### 2단계: 오른쪽 편집기에서 '끌어다 놓기' (가장 중요!)
이제 화면 **오른쪽**을 보세요. **'피벗 테이블 편집기'**라는 긴 막대가 생겼을 거예요. 여기서 **"B열(선물 종류)"**을 두 번만 끌어다 놓으면 됩니다.

1.  오른쪽 목록에서 **'선물 종류(B열 제목)'**를 마우스로 꾹 누르세요.
2.  그대로 끌어서 **[행]**이라고 적힌 박스 안에 놓으세요. (Drop!)
    *   *화면 변화: 왼쪽에 선물 이름들이 중복 없이 쭉 나열됩니다.*
3.  다시 한번 오른쪽 목록에서 **'선물 종류(B열 제목)'**를 마우스로 꾹 누르세요.
4.  이번에는 끌어서 **[값]**이라고 적힌 박스 안에 놓으세요. (Drop!)
    *   *화면 변화: 선물 이름 옆에 숫자가 생깁니다.*

#### 3단계: 완성된 표 확인하기
이제 왼쪽 표를 보세요.

*   **A열:** 선물 종류 이름들이 하나씩 적혀 있고,
*   **B열:** 그 선물을 선택한 사람이 몇 명인지 **숫자(COUNTA)**가 자동으로 계산되어 있습니다.

---

### 💡 요약
*   **[행]**에 놓으면: 항목의 **이름표**를 만들어 줍니다. (예: 사과, 배, 포도)
*   **[값]**에 놓으면: 그 항목의 **개수**를 세어 줍니다. (예: 3명, 5명, 2명)

어때요? 함수를 하나도 쓰지 않고 마우스 드래그 두 번만으로 통계가 끝났습니다! 천천히 따라 해 보시고 막히는 부분이 있으면 언제든 다시 물어봐 주세요.
