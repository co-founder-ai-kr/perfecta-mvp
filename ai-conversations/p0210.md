# 민순의 요청

```text
* 역할: 너는 이미지 인식을 통해 표를 작성하는 비즈니스 컨설턴트야.
* 상황: 손으로 그린 비즈니스 모델 캔버스 사진을 디지털 문서로 정리하고 싶어.
* 과업: 사진 속 내용을 분석해서 9개 블록으로 구성된 표로 정리해 줘.
* 제약: 모호한 글씨는 문맥에 맞게 보완해 줘.
* 출력: 9개 블록으로 구성된 표.
* 입력: [손으로 그린 BMC 사진 업로드]
```

# 제미나이의 응답

비즈니스 컨설턴트입니다. 현재 **[손으로 그린 BMC 사진]**이 업로드되지 않았습니다.

사진을 채팅창에 업로드해 주시면, 제가 이미지 속의 필기체를 인식하고 모호한 부분은 비즈니스 문맥에 맞게 보완하여 **아래와 같은 9개 블록 표**로 정리해 드리겠습니다.

분석을 원하시는 이미지를 업로드해 주세요.

---

### [예시 출력 양식: 비즈니스 모델 캔버스]

*(사진을 주시면 아래 표에 내용을 채워 드립니다)*

| 구분 | 상세 내용 |
| :--- | :--- |
| **1. 핵심 파트너 (Key Partners)** | • (공급업체, 파트너사 등 기입 예정)<br>• (외부 협력 관계 기입 예정) |
| **2. 핵심 활동 (Key Activities)** | • (비즈니스 운영을 위한 주요 활동 기입 예정)<br>• (제작, 플랫폼 개발 등) |
| **3. 핵심 자원 (Key Resources)** | • (물적, 지적, 인적, 재무적 자원 기입 예정) |
| **4. 가치 제안 (Value Propositions)** | • (고객에게 제공하는 핵심 가치 기입 예정)<br>• (해결하고자 하는 고객의 문제점) |
| **5. 고객 관계 (Customer Relationships)** | • (고객 확보 및 유지 전략 기입 예정)<br>• (개별 케어, 커뮤니티, 자동화 등) |
| **6. 채널 (Channels)** | • (고객에게 가치를 전달하는 경로 기입 예정)<br>• (웹사이트, 앱, 매장, SNS 등) |
| **7. 고객 세그먼트 (Customer Segments)** | • (목표 고객층 기입 예정)<br>• (틈새 시장, 대중 시장 등) |
| **8. 비용 구조 (Cost Structure)** | • (고정비, 변동비 등 주요 비용 기입 예정) |
| **9. 수익원 (Revenue Streams)** | • (수익 창출 방법 기입 예정)<br>• (판매, 구독료, 수수료 등) |

---
**💡 팁:** 사진이 흐릿하거나 글씨를 알아보기 힘들더라도, 전체적인 사업 맥락을 파악하여 최대한 정확하게 디지털 텍스트로 변환해 드립니다. **지금 사진을 올려주세요.**
