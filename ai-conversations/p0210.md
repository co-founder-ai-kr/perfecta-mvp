# 민순의 요청

```text
* 역할: 너는 이미지 인식을 통해 표를 작성하는 비즈니스 컨설턴트야.
* 상황: 손으로 그린 비즈니스 모델 캔버스 사진을 디지털 문서로 정리하고 싶어.
* 과업: 사진 속 내용을 분석해서 9개 블록으로 구성된 표로 정리해 줘.
* 제약: 모호한 글씨는 문맥에 맞게 보완해 줘.
* 출력: 9개 블록으로 구성된 표.
* 입력: [손으로 그린 BMC 사진 업로드]
```

# 제미나이의 응답

비즈니스 컨설턴트입니다. 의뢰하신 **손으로 그린 비즈니스 모델 캔버스(BMC)**를 디지털 문서로 변환할 준비가 되었습니다.

현재 **[손으로 그린 BMC 사진]**이 업로드되지 않았습니다. 사진을 업로드해 주시면, 이미지 속의 필체를 분석하고 모호한 부분은 비즈니스 문맥에 맞게 보완하여 아래와 같은 **9개 블록(9 Building Blocks)** 표로 정리해 드리겠습니다.

분석을 원하시는 이미지를 업로드해 주세요.

**(참고: 이미지를 업로드하시면 아래와 같은 형식으로 결과가 출력됩니다.)**
