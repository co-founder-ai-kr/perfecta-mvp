# 민순의 요청

```text
* 역할: 너는 구글 폼(Google Forms) 전문 튜터야.
* 상황: 잠재 고객 이메일 수집 및 구글 시트 자동 저장을 설정하려는데 메뉴 위치를 몰라 헤매고 있음.
* 과업:
  1. 구글 폼에서 새로운 설문을 만들고 싶은데 어디로 들어가야 하는지 안내.
  2. 이메일을 입력받으려면 질문 설정을 어떻게 바꿔야 하는지, 응답이 들어오면 자동으로 구글 시트에 저장되게 하려면 어떻게 연결해야 하는지 안내.
* 제약: "현재 화면의 메뉴 위치를 기준으로 설명해 줘"라고 명시.
* 출력: 따라 하기 쉬운 단계별 가이드(Step-by-step)
* 입력: [설문 조사의 목적]
 (예시: 선물 추천 서비스 사전 예약자 모집)
```

# 제미나이의 응답

반갑습니다! 구글 폼 전문 튜터입니다. 🎓

**"선물 추천 서비스 사전 예약자 모집"**을 위한 설문지를 만들고, 고객들의 이메일을 자동으로 엑셀(구글 시트)에 착착 쌓이게 하는 방법을 알려드릴게요.

화면의 **어느 위치**를 눌러야 하는지 정확히 짚어드릴 테니, 차근차근 따라와 주세요!

---

### 📌 1단계: 새로운 설문지 만들기
가장 먼저 백지상태의 설문지를 꺼내야겠죠?

1.  웹 브라우저 주소창에 `forms.google.com`을 입력해 접속하세요.
2.  화면 **왼쪽 상단**을 보세요. 알록달록한 십자가 모양(➕) 아이콘이 있고 그 아래 **[내용 없음]**이라고 적힌 버튼이 보일 거예요.
3.  그 **[내용 없음]** 버튼을 클릭하세요. 이제 새로운 설문지가 열립니다.
4.  화면 **왼쪽 맨 위** 구석에 `제목 없는 설문지`라고 적힌 글씨가 보이죠? 거기를 클릭하고 **"선물 추천 서비스 사전 예약"**이라고 이름을 바꿔주세요.

---

### 📌 2단계: 이메일 수집 설정하기 (가장 중요!)
이메일 주소를 입력받는 질문을 직접 만들 수도 있지만, 구글 폼의 **[설정]** 기능을 이용하면 이메일 형식이 맞는지 자동으로 검사해 줘서 훨씬 좋습니다.

1.  화면 **중앙 상단**을 보세요. 탭이 3개 있습니다. `[질문]` - `[응답]` - `[설정]` 순서로 되어 있죠?
2.  세 번째에 있는 **[설정]** 탭을 클릭하세요.
3.  맨 위에 **'응답'**이라는 메뉴가 보일 거예요. 그 옆의 작은 화살표(🔽)를 눌러 메뉴를 펼치세요.
4.  **'이메일 주소 수집'**이라는 항목이 보이죠? 기본값은 '수집 안함'으로 되어 있을 텐데, 이걸 클릭해서 **[응답자 입력]**으로 바꿔주세요.
    *   *이렇게 하면 설문지 맨 위에 '이메일' 입력란이 자동으로 생기고, 필수 항목으로 고정됩니다.*

---

### 📌 3단계: 구글 시트(엑셀) 자동 저장 연결하기
이제 고객이 응답 버튼을 누르는 순간, 자동으로 엑셀 표에 정리되도록 연결해 봅시다.

1.  다시 화면 **중앙 상단**의 탭을 보세요. 이번엔 가운데에 있는 **[응답]** 탭을 클릭하세요.
2.  화면 **오른쪽**을 살짝 보시면, **초록색 십자가 아이콘**(🟩)이 보일 거예요. (마우스를 올리면 'Sheets에 연결'이라고 뜹니다.)
3.  그 **초록색 아이콘**을 클릭하세요.
4.  팝업창이 뜨면 **'새 스프레드시트 만들기'**가 선택된 상태에서, 오른쪽 아래 **[만들기]** 버튼을 누르세요.
5.  새 창이 열리면서 엑셀 시트가 나타날 거예요. 이제부터 들어오는 모든 응답은 여기에 자동으로 저장됩니다! (이 창은 닫아도 계속 연결되어 있습니다.)

---

### 📌 4단계: 고객에게 링크 보내기
설정이 끝났으니 고객에게 보낼 주소를 따야겠죠?

1.  화면 **오른쪽 최상단**을 보세요. 보라색으로 된 **[보내기]** 버튼이 있습니다. 클릭하세요.
2.  팝업창 가운데에 **사슬 모양 아이콘**(🔗)이 있어요. (편지 봉투 아이콘 옆입니다.) 클릭하세요.
3.  인터넷 주소가 너무 기니까, 아래에 있는 **[URL 단축]** 체크박스를 체크하세요.
4.  주소가 짧아졌죠? 오른쪽의 **[복사]** 버튼을 누르세요.

---

**🎉 축하합니다!**
이제 복사한 링크를 잠재 고객들에게 공유하시면 됩니다. 고객이 응답을 제출할 때마다, 아까 연결해 둔 구글 시트에 이메일이 차곡차곡 쌓이는 것을 보실 수 있을 거예요.

혹시 진행하다가 막히는 부분이 있으면 언제든 다시 물어봐 주세요!
